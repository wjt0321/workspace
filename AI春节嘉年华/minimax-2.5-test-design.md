# MiniMax 2.5 模型能力测试 - 设计文档

## 总体目标

设计 **3 个独立的单文件（HTML）前端项目**，用于测试 MiniMax 2.5 模型的一次性代码生成能力。每个项目要求模型在 **单个 HTML 文件** 中完成所有功能（HTML + CSS + JavaScript），不依赖任何外部框架或构建工具。

> [!IMPORTANT]
> 本文档将作为给 MiniMax 2.5 的 **prompt 蓝图**，每个项目的描述都可以直接复制粘贴给模型作为输入。

## 项目 1：2048 小游戏

### 选型理由

2048 是一个经典的滑动合并益智游戏，能够全面测试模型的 **算法逻辑**（滑动合并规则）、**状态管理**（4×4 网格的精准操作）和 **动画能力**（方块移动/合并动效）。

### 功能需求

#### 核心玩法

- 4×4 网格棋盘
- 支持 **键盘方向键** 和 **触屏滑动** 操作
- 每次操作后随机生成一个新方块（90% 概率为 2，10% 概率为 4）
- 相同数字碰撞合并为两倍数字
- 无法继续移动时游戏结束，出现 2048 方块时显示胜利

#### UI 要求

- 深色主题背景（`#1a1a2e` 基调）
- 每个数字使用不同的渐变配色方案（2→浅色，2048→金色发光）
- 方块移动时有 **滑动动画**（`transition` 约 150ms）
- 方块合并时有 **弹跳缩放动画**
- 新方块出现时有 **淡入+缩放** 出现动画

#### 额外功能

- 实时分数显示
- 最高分记录（`localStorage` 持久化）
- "重新开始" 按钮
- 游戏结束/胜利的遮罩弹窗

### 验收标准

- [ ] 四个方向的滑动合并逻辑完全正确
- [ ] 动画流畅无卡顿
- [ ] 分数统计正确
- [ ] 游戏结束/胜利判定准确
- [ ] 移动端触屏操作可用

## 项目 2：Spotify 风格音乐播放器

### 选型理由

Spotify 的 UI 是现代 Web 应用设计的标杆，测试模型对 **复杂布局**（侧边栏+主内容区+底部播放栏）、**设计系统还原度** 和 **音频 API 操作** 的掌握程度。

### 功能需求

#### 布局结构

- **左侧边栏**：Logo、导航菜单（首页/搜索/媒体库）、播放列表区域
- **主内容区**：顶部渐变 banner + 歌曲列表（含编号、封面、曲名、艺术家、时长）
- **底部播放控制栏**：固定在底部，包含当前歌曲信息、播放控件、音量/进度条

#### UI 设计规范

- 采用 Spotify 经典深色主题（`#121212` 背景、`#1DB954` 绿色强调色）
- 侧边栏半透明效果（`rgba(0,0,0,0.7)`）
- 歌曲列表行 hover 时背景高亮
- 播放按钮为圆形绿色，hover 时放大
- 进度条和音量条支持拖动交互
- 响应式布局：窄屏时侧边栏折叠为图标模式

#### 音频功能

- 使用内置的 demo 音频数据（可用 `AudioContext` 生成简单音调）
- 播放/暂停/上一首/下一首
- 进度条实时更新，支持拖动跳转
- 音量滑块控制
- 歌曲列表点击切换播放

#### 数据模拟

- 至少 **8 首** 模拟歌曲数据（曲名、艺术家、时长、封面色块）
- 至少 **3 个** 播放列表分类

### 验收标准

- [ ] 布局高度还原 Spotify 风格
- [ ] 播放控件功能完整可用
- [ ] 进度条拖动流畅
- [ ] 歌曲切换逻辑正确
- [ ] 窄屏响应式正常

## 项目 3：太空探索主题动态网页

### 选型理由

太空主题是展示 **CSS 动画能力**、**Canvas/粒子效果** 和 **视觉创意** 的绝佳场景，全面测试模型的前端视觉表现力。

### 功能需求

#### 视觉效果

- **星空背景**：使用 Canvas 或纯 CSS 实现数百个闪烁星星的动态星空
- **流星效果**：随机出现的流星划过屏幕（带尾迹渐变）
- **行星系统**：至少展示 3 个行星的公转动画（不同轨道、不同速度）
- **视差滚动**：滚动时多层背景以不同速度移动，营造纵深感

#### 页面内容（单页滚动式）

1. **Hero 区域**：大标题 "Explore the Cosmos" + 粒子星空 + 向下滚动引导箭头动画
2. **行星展示区**：3-4 个行星卡片，hover 时放大并显示详细信息（名称、质量、特点），带 3D 翻转或浮起效果
3. **宇宙时间线**：用纵向时间线展示宇宙大事件（大爆炸→星系形成→太阳系诞生→现在）
4. **页脚**：深空渐变 + 社交链接图标

#### 动画与交互

- 所有元素使用 `IntersectionObserver` 实现 **滚动触发动画**（淡入、滑入、缩放等）
- 鼠标移动时星空产生轻微的 **视差偏移**
- 行星卡片 hover 有 **发光光环** 效果（`box-shadow` 动画）
- 整体使用 **平滑滚动**（`scroll-behavior: smooth`）
- 加载时有 **入场动画序列**（标题逐字显现 + 星空展开）

#### 配色方案

- 主底色：深空黑 `#0a0a1a` → `#0d1b2a` 渐变
- 强调色：星辉蓝 `#00d4ff`、星云紫 `#7b2ff7`
- 文字色：`#e0e0e0` 主文字、`#88ccff` 高亮
- 渐变：大量使用径向/线性渐变模拟宇宙氛围

### 验收标准

- [ ] 星空粒子效果流畅运行
- [ ] 流星随机出现且视觉效果自然
- [ ] 行星公转动画正确
- [ ] 视差滚动效果明显
- [ ] 滚动触发动画按预期触发
- [ ] 页面整体视觉震撼
