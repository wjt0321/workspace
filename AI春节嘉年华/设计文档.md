## 一、项目概述

### 1.1 项目背景

- **活动名称**：AI春节嘉年华
- **活动时间**：2026年1月16日 - 2月20日（春节前后）
- **参与人数**：100+ 朋友
- **核心目标**：通过AI技术打造一个视觉惊艳、互动有趣、让100位朋友都愿意分享到朋友圈的春节活动

### 1.2 核心功能

```
┌─────────────────────────────────────────────────────────────┐
│                    四大核心模块                              │
├─────────────┬─────────────┬─────────────┬───────────────────┤
│ 🎊 娱乐互动  │ 🎇 文化传承 │ 📔 纪念册生成 │ 🔗 社群联动        │
├─────────────┼─────────────┼─────────────┼───────────────────┤
│ • AI红包问答 │ • AI春联    │ • AI年度故事 │ • 新年愿望墙       │
│ • 默契大考验 │ • 灯谜会    │ • 朋友词云   │ • 祝福语接龙       │
│ • 真心话地图 │ • 知识挑战  │ • 照片故事   │ • 马年抽签         │
└─────────────┴─────────────┴─────────────┴───────────────────┘
```

### 1.3 技术栈

- **前端**：Vue 3 + Vite + Tailwind CSS + GSAP
- **后端**：Node.js + Express + n8n
- **AI**：Claude Code API（主力） + GLM-4 API（备用）
- **部署**：NAS + Cloudflare 隧道
- **存储**：NAS本地存储 + SQLite

> [!note] **重要更新**
> 由于用户只有GLM-4.7（智谱）API，AI主力已改为GLM-4.7

## 三、UI页面详细设计

### 3.1 首页设计

**设计目标**：第一眼就惊艳，让用户想继续探索

#### 动画流程

1. 粒子从底部升起 → 汇聚成马年标识
2. 金色光晕扩散 → 出现标题
3. 背景音乐 → 音量渐入（可选）
4. 点击按钮 → 粒子礼花绽放 → 页面翻转进入主页

#### 技术要点

- **首屏加载**：必须控制在2秒内
- **动画流畅度**：60fps
- **适配**：iPhone SE到Pro Max

### 3.2 主菜单设计

四大模块卡片：
- 🎊 娱乐互动
- 🎇 文化传承  
- 📔 纪念册生成
- 🔗 社群联动

### 3.3 核心模块

#### A. AI春联生成器

输入关键词 → AI生成3副春联 → 选择喜欢的版本 → 生成海报 → 分享朋友圈

#### B. AI年度故事（核心重头戏）

加载动画（星辰汇聚）→ 关键词展示 → 高光时刻 → 朋友评价 → 2026预测 → 分享
