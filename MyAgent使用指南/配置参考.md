# 配置参考

> 完整的配置项说明。配置文件位于 `~/.myagent/`。

---

## 配置文件结构

```
~/.myagent/
├── config.yaml    # Agent 核心配置
├── gateway.yaml   # 网关平台配置
└── .env           # API Keys 和密钥
```

## config.yaml（Agent 核心配置）

```yaml
# 默认模型
model: anthropic/claude-sonnet-4

# 上下文压缩阈值（轮次）
context_max_turns: 50

# 记忆配置
memory:
  auto_extract: true
  rag_threshold: 0.7

# 工具配置
tools:
  allowed:
    - Bash
    - Read
    - Write
    - Edit
    - Glob
    - Grep
    - WebFetch
    - WebSearch
    - ImageAnalyze
    - TextToSpeech
    - Todo
    - Git
  disallowed: []

# Agent 默认配置
agent:
  effort: medium
  max_turns: 50
  permission_mode: ask  # ask | dontAsk | approve

# 工作区
workspace:
  root: ~/.myagent/workspace
```

## gateway.yaml（多通道网关配置）

```yaml
# 启用哪些平台
platforms:
  - telegram
  - discord
  - slack
  - feishu
  - github

# 会话隔离策略
session:
  per_user: true
  per_group: true
  per_thread: true

# 权限策略
permission:
  default: ask          # ask | allow | deny
  allowlist:
    - ou_xxx           # 飞书 open_id
    - user_xxx         # Telegram user_id
  deny_list: []

# 各平台独立配置
telegram:
  bot_token: ${TELEGRAM_BOT_TOKEN}
  allowed_users: []

discord:
  bot_token: ${DISCORD_BOT_TOKEN}
  guild_id: ...

slack:
  bot_token: ${SLACK_BOT_TOKEN}
  app_token: ...

feishu:
  app_id: ${FEISHU_APP_ID}
  app_secret: ${FEISHU_APP_SECRET}
  bot_name: MyAgent

github:
  webhook_secret: ${GITHUB_WEBHOOK_SECRET}
  app_id: ${GITHUB_APP_ID}
  token: ${GITHUB_TOKEN}
```

## 配置优先级

```
环境变量 > .env > config.yaml / gateway.yaml
```

`.env` 中使用 `${VAR}` 语法引用环境变量。
