# 部署指南

---

## Docker（推荐本地/小规模部署）

### 构建镜像

```bash
git clone https://github.com/wjt0321/MyAgent.git
cd MyAgent
docker build -t myagent .
```

### 运行 Web UI

```bash
docker run -d \
  --name myagent \
  -p 8000:8000 \
  -v myagent-data:/app/data \
  -e ANTHROPIC_API_KEY=$ANTHROPIC_API_KEY \
  myagent
```

### Docker Compose（多服务）

```bash
# 启动 Web UI
docker compose up -d web

# 启动 Web + Bot（Gateway）
docker compose --profile bot up -d
```

### 挂载配置

```bash
docker run -d \
  -p 8000:8000 \
  -v /path/to/.myagent:/app/.myagent \
  -e MYAGENT_HOME=/app/.myagent \
  myagent
```

**注意**：TUI 适合本地终端使用，不建议作为容器默认进程。

## Kubernetes（Helm Chart）

```bash
helm install myagent ./deploy/helm/myagent \
  --set myagent.apiKeys.anthropic="your-api-key" \
  --set myagent.provider="anthropic"
```

### 完整 Helm 配置示例

```yaml
# values.yaml
myagent:
  replicaCount: 2

  image:
    repository: myagent
    tag: latest
    pullPolicy: IfNotPresent

  apiKeys:
    anthropic: ""        # 填入你的 Key
    openai: ""
    deepseek: ""

  provider: anthropic
  model: claude-sonnet-4-20250514

  persistence:
    enabled: true
    storageClass: "standard"
    accessMode: ReadWriteOnce
    size: 5Gi

  ingress:
    enabled: true
    host: myagent.example.com
    tls: true

  resources:
    limits:
      cpu: 2000m
      memory: 4Gi
    requests:
      cpu: 500m
      memory: 1Gi

  metrics:
    enabled: true        # 开启 Prometheus metrics
```

### Prometheus + Grafana 监控

```bash
# Prometheus Operator 已安装前提下
helm install myagent ./deploy/helm/myagent \
  --set myagent.metrics.enabled=true

# 查看 Grafana Dashboard
kubectl port-forward svc/grafana 3000:3000
# 导入 deploy/grafana/dashboard.json
```

## systemd（服务器常驻）

```bash
# /etc/systemd/system/myagent.service
[Unit]
Description=MyAgent Web UI
After=network.target

[Service]
Type=simple
User=your-user
WorkingDirectory=/opt/myagent
Environment="ANTHROPIC_API_KEY=sk-ant-..."
ExecStart=/usr/local/bin/myagent web --port 8000
Restart=always
RestartSec=5

[Install]
WantedBy=multi-user.target
```

```bash
sudo systemctl enable myagent
sudo systemctl start myagent
sudo journalctl -u myagent -f
```

## 生产环境检查清单

| 检查项 | 说明 |
|--------|------|
| ✅ HTTPS/SSL | 用 Nginx/Caddy 反向代理，Web UI 必须 HTTPS |
| ✅ API Keys | 用环境变量或 Vault，不要写进镜像 |
| ✅ 防火墙 | 只暴露 8000(Web)/18789(Gateway) |
| ✅ 日志轮转 | 配置 logrotate，避免磁盘爆满 |
| ✅ 监控 | 开启 Prometheus metrics，Grafana 看板 |
| ✅ 备份 | 定期备份 `~/.myagent/sessions/` 和 `memory/` |
| ✅ 安全加固 | 启用 JWT 认证，配置 allowlist |

## 日志格式

支持结构化 JSON 日志（`--json-log`），方便日志收集：

```bash
myagent web --json-log | jq .
```

字段：

```json
{
  "level": "INFO",
  "ts": "2026-04-26T01:00:00Z",
  "msg": "Session created",
  "session_id": "abc123",
  "user_id": "ou_xxx"
}
```

## Prometheus Metrics

启用后暴露 `/metrics` 端点，包含：

- `myagent_sessions_total` — 会话总数
- `myagent_tool_calls_total` — 工具调用次数
- `myagent_llm_tokens_total` — Token 消耗
- `myagent_task_duration_seconds` — 任务耗时
- `myagent_gateway_messages_total` — 网关消息量（按平台分）
