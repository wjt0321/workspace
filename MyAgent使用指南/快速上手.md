# 快速上手

> 从零安装到第一个对话，5 分钟搞定。

---

## 前置要求

- Python 3.11+
- pip
- 至少一个 LLM API Key（推荐 Anthropic Claude）

---

## 安装

```bash
pip install myagent
```

或开发模式：

```bash
git clone https://github.com/wjt0321/MyAgent.git
cd MyAgent
pip install -e ".[dev]"
```

---

## 初始化

**快速模式**（生成最小配置）：

```bash
myagent init --quick
```

**完整向导**（交互式引导）：

```bash
myagent init
```

初始化会创建：

```
~/.myagent/
├── config.yaml          # Agent 配置
├── gateway.yaml         # 网关平台配置
├── .env                 # API Keys（需手动填写）
├── sessions/            # 会话存储
├── logs/                # 日志
└── workspace/           # Agent 工作区
```

---

## 填写 API Key

编辑 `~/.myagent/.env`，例如：

```bash
# LLM（国际）
ANTHROPIC_API_KEY=sk-ant-...

# 或国内
ZHIPU_API_KEY=...
MOONSHOT_API_KEY=...
MINIMAX_API_KEY=...
DASHSCOPE_API_KEY=...
```

---

## 启动方式

### Web UI（推荐新手）

```bash
myagent web --port 8000
```

然后浏览器打开 http://localhost:8000

### TUI（终端界面）

```bash
myagent --tui
```

### Gateway（后台服务）

```bash
myagent gateway --port 18789
```

### CLI 诊断

```bash
myagent doctor   # 检查配置状态，提示下一步
```

---

## 首次体验路线

1. 打开 Web UI，停留在欢迎页 → 查看 **3步快速上手**
2. 点击 **Recommended Demo Path** 卡片 → 体验 Chat → Tasks → Workspace
3. 从会话控制栏切换 `agent` 或 `model` → 观察实时会话摘要变化
4. 输入 `/plan <任务描述>` → 观察 Task → Team → Review 全流程

---

## Web UI 功能一览

| 区域 | 说明 |
|------|------|
| 左侧导航 | Chat / Tasks / Files / Workspace / Team |
| Ctrl+K | 命令面板，快速跳转 |
| `/plan` | 创建任务，进入 Plan→Execute→Review |
| `/agent` | 切换 Agent 类型 |
| `/model` | 切换 LLM 提供商 |
| `/session` | 查看/切换会话 |
| 右侧面板 | 工具卡片、任务详情、会话摘要 |

---

## TUI 功能一览

- 状态侧边栏：会话、任务、内存概览
- 斜杠命令：`/plan`、`/agent`、`/model`、`/session`
- 模态审批：任务权限申请弹窗
- 任务面板：进度、活跃 Agent、时间线
