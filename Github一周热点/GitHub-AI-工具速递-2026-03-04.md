# GitHub AI 工具与项目速递 — 2026-03-04

> [!abstract] 本期摘要
> 本周GitHub AI领域迎来多项重磅更新。**Hermes Agent** 全开源个人AI代理首发，**MassGen** 多智能体协作框架发布v0.1.56，**Alibaba OpenSandbox** 生产级沙箱平台开源。本期重点关注：个人AI代理、多智能体协作、Agent Skills标准化三大领域的突破性进展。

## 📊 本周热点趋势概览

### 核心热点事件

#### 1. Hermes Agent 发布 — 全开源个人AI代理

> [!success] 里程碑发布
> **Nous Research** 推出的全开源AI代理，可安装在服务器上，持续学习你的项目、构建自己的技能、定时运行任务，并通过Telegram/Discord/Slack/WhatsApp等多平台与你保持联系。

- **核心创新**：持久化记忆 — 代理会记住你的偏好、项目和环境，越用越聪明
- **多平台支持**：Telegram、Discord、Slack、WhatsApp、CLI 全平台覆盖
- **技能自增长**：解决难题后会自动编写技能文档，支持 `agentskills.io` 开放标准
- **安全沙箱**：支持 Local、Docker、SSH、Singularity、Modal 五种终端后端

```bash
# 安装命令 (Linux/macOS)
curl -fsSL https://raw.githubusercontent.com/NousResearch/hermes-agent/main/scripts/install.sh | bash

# Windows (PowerShell)
irm https://raw.githubusercontent.com/NousResearch/hermes-agent/main/scripts/install.ps1 | iex
```

#### 2. MassGen v0.1.56 发布 — 多智能体协作框架

> [!info] 版本更新
> **MassGen** 是一个多智能体扩展系统，通过冗余和迭代精炼协调AI代理解决复杂任务。每个代理处理完整问题，观察、批评并建立在彼此工作之上。

**v0.1.56 新特性：**
- **📋 Spec Plan Mode** — `plan_mode="spec"` 用于执行前进行正式需求规格说明
- **🎯 ask_others Targeting** — `target_agents` 参数实现定向代理间通信
- **🔍 Critic Subagent** — 新增子代理类型，用于诚实、无偏见的质量评估

```bash
# 安装
pip install massgen

# 快速启动
massgen --quickstart
```

#### 3. Alibaba OpenSandbox 开源 — 生产级AI代理沙箱

> [!quote] 企业级开源
> **阿里巴巴**开源的通用沙箱平台，为AI应用提供多语言SDK、统一沙箱API，以及Docker/Kubernetes运行时。

- **安全隔离**：生产级基础设施，安全运行不可信的代理代码
- **多语言SDK**：支持多种编程语言
- **容器化部署**：原生支持Docker和Kubernetes
- **浏览器自动化**：内置浏览器自动化支持
- **网络隔离**：完善的网络隔离机制

**什么是Agent Skills？**

Agent Skills是模块化、标准化的 `SKILL.md` 包，通过渐进式披露为代理提供按需能力：
- 轻量级元数据可早期加载
- 完整指令仅在相关时加载
- 支持资源按需访问

**支持的平台：**
- Claude Code
- OpenAI Codex
- Gemini CLI
- Cursor
- VS Code
- GitHub Copilot
- Mistral Vibe
- Manus
- OpenCode

> [!success] 亮点
