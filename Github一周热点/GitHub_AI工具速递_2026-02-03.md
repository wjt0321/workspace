**简短摘要**: 微软开源的全球首个 1-bit LLM 推理框架，通过三元权重 {-1, 0, 1} 实现极致量化，让 CPU 也能高效运行大模型。

> [!success] 亮点 (Why interesting)
> - **颠覆性架构创新**: 将传统 FP16/INT8 乘法运算转化为简单加减法
> - **极致性能提升**: CPU 推理速度提升 2-6 倍，能耗降低 70%+
> - **超小模型体积**: 2B 参数模型仅 0.4GB，100B 模型可在消费级 CPU 运行
> - **端侧 AI 里程碑**: 为手机、笔记本等边缘设备运行大模型铺平道路
> - **开源生态友好**: 支持 Llama3、Falcon3 等主流模型架构

> [!warning] 风险与注意
> - 当前仅支持推理，训练功能尚在开发中
> - 精度损失约 2-5%，不适合对精度要求极高的场景
> - 需要专门的模型转换工具，不能直接使用现有模型

**重要文件**:
- `bitnet.cpp` - 核心推理引擎
- `setup_env.sh` - 环境配置脚本
