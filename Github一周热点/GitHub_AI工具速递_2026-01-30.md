## 🔍 关注趋势

本期观察到的关键趋势：

1. **AI Agent 框架爆发式增长**: 多智能体系统成为热点，LangChain、AutoGen、MetaGPT、CrewAI 等框架百花齐放，反映出从单一 AI 助手向协作式智能体系统演进的趋势。
2. **本地化与隐私优先**: Moltbot 的爆火体现了开发者对本地部署、数据隐私的强烈需求。自托管 AI 助手成为新的技术方向。
3. **TypeScript 在 AI 工具中崛起**: TypeScript 在 2025 年 8 月首次超越 Python 成为 GitHub 上使用最广泛的语言，在 AI 工具开发中的占比显著提升。
4. **RAG 技术成为标配**: LlamaIndex、Haystack 等专注于 RAG 的框架持续增长，数据驱动的 AI 应用需求旺盛。
5. **开源超越商业产品**: OpenCode 超越 Claude Code 成为最受欢迎的 AI 编程工具，标志着开源社区在 AI 工具领域的创新能力已经可以挑战商业巨头。
