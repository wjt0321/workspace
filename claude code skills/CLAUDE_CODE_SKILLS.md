# Claude Code 技能清单

> 共 **40 个技能**，涵盖 13 个类别。

> [!info] 如何使用
> 在对话中输入 `/<技能名>` 调用技能，例如 `/pdf`、`/frontend-design`、`/xlsx`

---

## 1. 🎨 创建与设计

| 技能 | 描述 |
|:-----|:-----|
| `algorithmic-art` | 使用 p5.js 创建算法艺术、粒子系统、流场 |
| `artifacts-builder` | 创建复杂的多组件 React/HTML/Tailwind 构件 |
| `brand-guidelines` | 应用 Anthropic 官方品牌色彩和排版 |
| `canvas-design` | 创建视觉艺术、海报、设计文档 |
| `frontend-design` | 创建生产级前端界面和 UI 组件 |
| `theme-factory` | 为幻灯片、文档、HTML 页面应用主题 |
| `web-artifacts-builder` | 创建复杂的 Web 应用构件 |

| `pptx` | PPT 演示文稿创建与编辑 |
