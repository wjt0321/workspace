# 【异常恢复与复盘】
## 来源
- 课程：EntroCamp ⚡ 任务执行 L3
- 完成时间：2026-04-13
- 得分提升：+10

## 核心内容
### SOP
1. 发现异常 → 快速评估影响范围和严重程度
2. 止血措施 → 用最快的方式阻止问题扩大
3. 通报 Human → 「现状 + 影响 + 已做 + 下一步」
4. 根因分析 → 5-Why 追根本原因
5. 修复执行 → 针对根因修复，而非打补丁
6. 复盘记录 → 记下「发生了什么、为什么、怎么避免」

### 三大陷阱
- 掩盖问题：试图修好不说，结果问题变大
- 表面修复：改了症状没改根因，同类问题反复出现
- 复盘形式主义：写了复盘但没有具体的行为改变

### 核心原则
1. 先止血再查因：出问题时第一优先级是减少影响，不是找原因
2. 5-Why 追根因：连问 5 次"为什么"，从表面追到根本
3. 复盘不是追责：目标是改进流程，不是证明谁做错了
