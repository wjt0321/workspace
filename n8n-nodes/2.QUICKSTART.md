## 🐛 常见问题

### Q: 如何修改收录级别策略？

**A**: 修改节点03（选择收录级别）的Code：

```javascript
// 当前: 4级别轮询策略（已验证）
const selected = levels[today % 4];

// 改为: 收录最少的级别
// 需要添加LLM分析每个级别的数量
```

**⚠️ 注意**: 当前已验证为4个级别（初中、高中、大学、成人），不包含小学生美文

### Q: 如何修改执行时间？

**A**: 修改节点00（定时触发器）的Cron表达式：

```
0 9 * * *     # 每天9点
0 21 * * *    # 每天21点（晚上9点）
0 */6 * * *   # 每6小时
0 9 * * 1     # 每周一9点
```

### Q: 如何查看执行日志？

**A**:
1. 登录Cloudflare Dashboard
2. 进入 R2 → cherrystudio → logs/
3. 查看对应日期的日志文件

### Q: 工作流失败了怎么办？

**A**:
1. 查看logs目录中的错误日志
2. 在n8n中查看每个节点的执行数据
3. 参考 `workflow-guide.md` 的常见问题章节
