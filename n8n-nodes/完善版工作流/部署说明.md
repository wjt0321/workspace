# 📋 美文收集自动化工作流 - 部署说明

## 🚀 快速部署（5分钟）

### 1. 导入工作流
1. 打开n8n → 创建新工作流
2. 点击右上角 `...` → `Import from File`
3. 选择 `美文收集完整工作流.json`

### 2. 配置凭证

#### Cloudflare R2 凭证
```
类型：AWS S3（兼容模式）
Access Key ID：<你的R2 Access Key>
Secret Access Key：<你的R2 Secret Key>
Custom Endpoint：https://<ACCOUNT_ID>.r2.cloudflarestorage.com
Region：auto
```

获取方式：Cloudflare Dashboard → R2 → Manage R2 API Tokens

#### DeepSeek 凭证
导入后直接选择你已配置的 `DeepSeek` 凭证即可。

> 节点使用 OpenAI 兼容模式，会自动连接到你的 DeepSeek 凭证。

### 3. 激活工作流
点击左上角 `Inactive` → 切换为 `Active`

---

## 📊 工作流结构

```
定时触发(9:00) → 读取总索引 → 选择级别 → LLM生成片段
    ↓
提取数据 → 读取片段文件 → 更新片段 → 上传片段
    ↓
读取索引 → 更新索引 → 上传索引 → LLM搜索原文
    ↓
解析结果 → 版本检查 ─┬→ 读取模版 → 格式化 → 上传文章
                     └→ 记录失败（终止）
    ↓
读取总索引 → 更新总索引 → 上传总索引 → 生成日志 → 上传日志
```

## 🔧 原方案修复内容

| 问题 | 修复方式 |
|------|----------|
| Binary数据传递失败 | 使用Code节点处理Buffer转换 |
| 节点引用语法错误 | 修正为 `$('节点名').first().json` |
| LLM输出解析失败 | 添加JSON提取和fallback处理 |
| 缺少完整工作流 | 提供一键导入JSON |

## ⚠️ 注意事项

1. **首次运行**：建议手动点击 `Execute Workflow` 测试
2. **凭证名称**：需与JSON中一致（`Cloudflare R2`、`智谱AI`）
3. **存储桶**：确保R2中有 `cherrystudio` 存储桶
4. **文件结构**：确保索引文件已上传到R2
