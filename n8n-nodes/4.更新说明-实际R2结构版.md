## 🔑 关键配置更新

### 节点03: 选择收录级别

**更新前**: 只有snippetFile和indexFile
**更新后**: 添加了templateFile配置

```javascript
const levels = [
  {
    key: '小学生美文',
    name: '小学生美文',
    snippetFile: '小学生美文/小学生美文精选100篇（完整版）.md',
    indexFile: '小学生美文/美文收集索引.md',
    templateFile: '小学生美文/美文赏析与教学通用模版.md'  // ✅ 新增
  },
  // ... 其他级别
];
```

### 节点14: 读取模版

**更新前**: `templates/{{级别}}模版.md`
**更新后**: `{{ $('选择收录级别').item.json.templateFile }}`

这样可以直接从各级别文件夹下读取对应的模版文件。
