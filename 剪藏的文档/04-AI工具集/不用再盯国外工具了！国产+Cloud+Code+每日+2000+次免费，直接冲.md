## 安装 nodejs

打开 nodejs 官方下载地址：https://nodejs.org/en/download

![img](https://mmbiz.qpic.cn/sz_mmbiz_png/K0sEmT4Gchd8jbfC9dUA99MuV5GWqjjj9SzsHegneWg3AejNIYnx05pjFJiaTmMrEibwzPwYxC5llYgJZ3RibSzibw/640?wx_fmt=png&from=appmsg#imgIndex=1)

img

根据自己的情况，选择版本，点击下载安装包。版本选择 **v14.17.0 后的版本。**

## 安装 Qwen

安装 nodejs 之后，就可以开始安装 Qwen-Code 了！

输入命令：

```Plain Text
npm install -g @qwen-code/qwen-code@latest
qwen --version
```

命令执行完成，就安装完成了，安装完成之后会显示版本信息。

启动软件：

```Plain Text
qwen
```

直接输入 qwen 就可以启动了。

## 使用 Qwen

在需要保存项目的路径中打开 cmd，然后输入 qwen。

具体操作步骤：

1. 在文件夹空白处按住 Shift 键 + 鼠标右键，选择 “在此处打开命令窗口” 或 “在此处打开 PowerShell 窗口”（Win10/11 默认可能是 PowerShell，可直接输入 cmd 切换到命令提示符）。

2. 待 cmd 窗口打开后，直接输入 “qwen” 并按回车即可。
