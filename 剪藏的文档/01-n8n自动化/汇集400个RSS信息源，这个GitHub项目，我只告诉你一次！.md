# 汇集400个RSS信息源，这个GitHub项目，我只告诉你一次！

Original 鲤鱼冲啊 [鲤鱼冲啊](javascript:void(0);)*2025年10月15日 22:49* *福建*

一个链接，终结你的信息焦虑！这是一个GitHub神级项目，它像一个“情报总局”，已经帮你收集好了400个AI、科技、产品领域的“情报源头（RSS）”，让你从此只看一手猛料！

不了解RSS的，可以先看看这篇：[信息孤岛终结者FOLO：把公众号、B 站、油管，X搬进一个“信息指挥部”](https://mp.weixin.qq.com/s?__biz=Mzg5ODU1NjQ3MA==&mid=2247484714&idx=1&sn=0ea5ace2faeb217425474188b7922cad&scene=21#wechat_redirect)。

FOLO虽好，但rss源需要自行搜集，而公众号因为封闭的原因，是最难收集到的，还经常失效。所以这个项目起到很大帮助。

项目名称为BestBlogs，当前有1.4k星，可以邮件订阅精选推送也可以在线直接阅读，GitHub链接：https://github.com/ginobefun/BestBlogs；
