## 2.n8n获取公众号文章，插入飞书的整体思路

首先，获取公众号Biz号，确保能准确标识公众号。然后在n8n中搭建一个工作流，流程清晰如下：

- **手动触发器：**通过手动启动按钮触发整个流程，便于调试和快速启动。

- **HTTP请求节点（获取公众号历史文章）**：发送请求到API，获取公众号历史文章数据。

- **Split Out节点（拆分文章）：**将获取到的历史文章数组拆分成单篇文章，便于逐一处理。

- **HTTP请求节点（获取文章详情）：**根据文章链接，通过API获取完整的文章内容。

- **Edit Field节点（提取关键信息）：**从返回的数据中提取所需字段，如标题、内容、发布时间等。

- **Code节点（处理非法字符）：**处理文章中的非法字符，确保数据格式正确，避免插入到飞书时出错。

- **飞书节点（上传到飞书多维表格）：**将清理后的文章内容通过飞书节点上传到飞书多维表格中，便于管理和查看。

## 3.实现流程

### 第一步：创建触发节点

首先，点击“+”号，选择“手动触发”节点，并将其添加到工作流中。

### 第二步：获取公众号Biz号

公众号的Biz号是微信为每个公众号分配的唯一标识符，用于文章采集和链接构造等操作。

获取Biz号的方法：打开公众号的文章页面，右键点击页面，选择“查看网页源码”或者直接按“Ctrl+U”。

然后在源码中搜索“biz”，它就是公众号的唯一标识。

![Image 3](https://mmbiz.qpic.cn/mmbiz_png/b5KpQwrFYRrHEpdBptBibX8FpWfbdCfIdyHKrxkdZibYS72WXcVicibE9OmoV4V5wic8qLsSCjQjnicLrzlZQ84jRzkQ/640?wx_fmt=png&from=appmsg&watermark=1#imgIndex=3)

### 第八步：提取关键信息

### 第九步：处理内容中的非法字符

公众号文章内容中往往有一些非法字符（如换行符、回车符、制表符等），这些会影响数据的正常处理。使用“Code”节点来清理这些字符，确保内容格式正确，能够顺利插入到飞书多维表格中。

![Image 11](https://mmbiz.qpic.cn/mmbiz_png/b5KpQwrFYRrHEpdBptBibX8FpWfbdCfId5x2dNicgjvRFxLCwvfIiav0un89gaDicnhbDPTlBBmvEB8MsD0fic0uDmg/640?wx_fmt=png&from=appmsg&watermark=1#imgIndex=11)

```Plain Text
return items.map(item => {

  let txt = item.json.content;
  /* 1.  invisible control chars  （0x00-0x1F 除 \t\n\r 之外）*/
  txt = txt.replace(/[\x00-\x08\x0B\x0C\x0E-\x1F]/g, '');
  /* 2.  换行/回车/制表 → 空格，合并连续空白 */
  txt = txt.replace(/[\r\n\t]+/g, ' ').replace(/\s{2,}/g, ' ').trim();
  /* 3.  转义 JSON 敏感字符，防止后续拼 body 时破 JSON */
  txt = txt
    .replace(/\\/g, '\\\\')   // 反斜杠
    .replace(/"/g, '\\"')    // 双引号
    .replace(/\//g, '\\/');  // 正斜杠（可选）
  
  return {
    json: {
      title: item.json.title,
      nick_name: item.json.nick_name,
      create_time: item.json.create_time,
      url: item.json.url,
      content: txt           
    }
  };
});
```
