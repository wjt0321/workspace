# 📋 美文收集标准操作程序 (SOP)

## 📅 版本信息
- **版本**：1.0.0
- **创建日期**：2025-12-21
- **适用范围**：小学生美文、初中生美文、高中生美文、大学生美文、成人美文
- **制定依据**：严格按照`.skills`目录下的skill流程制定
- **使用软件**：Obsidian

## 🚨 核心原则声明

### ⚠️ 绝对遵守条款
1. **严格遵循9步工作流程**：必须按照Step 0→Step 1→Step 2→Step 3→Step 4→Step 5→Step 6→Step 7→Step 8的顺序执行
2. **不可跳过任何步骤**：每一步都必须完成，不得遗漏（Step 0预查重不可跳过）
3. **不可颠倒顺序**：必须严格按序执行，不得打乱顺序（Step 0必须最先执行）
4. **不可同时执行**：必须按步骤顺序依次完成，不得同时进行多个步骤
5. **不可做任何更改**：严格按照skill描述执行，不得擅自修改或简化流程
6. **不可跳过步骤**：任何跳过行为都视为操作违规（Step 0跳过视为严重违规）
7. **🚨 Step 8铁律**：总索引更新为不可违背的强制步骤，任何收录工作都必须以此结束

### 🎯 执行要求
- **优先级最高**：此步骤必须在任何其他步骤之前执行
- **必须检查总索引**：仔细阅读`美文总索引.md`的完整内容
- **预判重复风险**：了解各文件夹现有文章，避免收集重复内容
- **记录潜在重复**：识别可能重复的作品，为后续去重做准备

### 📂 检查内容

- **作家分布**：了解已收录作家的作品分布
- **题材覆盖**：查看现有题材分类，避免过度集中
- **重复关系**：查看跨文件夹重复文章的记录
- **数量统计**：了解各文件夹的完成度

### ✅ 预查重确认

- [ ] 已识别现有文章分布情况
- [ ] 已标记潜在重复风险作品
- [ ] 已规划非重复文章收集策略
- [ ] 确认不会故意收集重复文章

### 🚨 重要提醒
- **此步骤不可跳过**：必须在Step 1之前完成
- **查重步骤仍需执行**：此预查不替代Step 5的正式查重
- **减少重复概率**：通过预查大幅降低重复收集的可能性
- **提升工作效率**：避免无效收集，提高工作质量

## 📋 Step 1️⃣: 收集片段 (prose-snippet-collector) - **执行前必须完成Step 0**

### 🎯 执行要求
- **必须调用skill**：`prose-snippet-collector/SKILL.md`
- **严格按skill执行**：不得修改skill描述的任何要求
- **片段长度标准**：
  - 小学生：150-300字
  - 初中生：200-400字
  - 高中生：300-500字
  - 大学生：300-500字
  - 成人：300-500字
- **开头Emoji要求**：必须添加2-3个主题相关Emoji组合
- **正文穿插Emoji**：每2-3句话1个，与内容高度贴合（所有级别统一标准）
- **句子定义**：以任何标点符号结尾的完整语义单元
- **作家范围**：必须来自近现代中文散文大家

### 📂 输出文件
- `d:\workspace\{级别}美文\{级别}美文精选100篇.md` 或续辑
- 格式必须100%符合skill示例

## 📋 Step 4️⃣: 格式化文章 (prose-article-formatter)

### 🎯 执行要求
- **必须调用skill**：`prose-article-formatter/SKILL.md`
- **必须先读取模版**：`d:\workspace\{级别}美文\美文赏析与教学通用模版.md`
- **100%符合模版要求**：
  - 完整的YAML元数据（title、source、created、tags）
  - H1标题（含Emoji+完整版标注）
  - H3作者署名
  - 原文正文（开头2-3个Emoji，H3+Emoji分段，正文穿插Emoji）
  - 所有核心板块（根据级别要求）
  - 所有必需Emoji
- **文件命名**：`{序号}.{作品名}-{作者}.md`

### � 模版使用规范（避免阅读混乱）
- **区分占位符与操作指南**：
  - **占位符**：`{{...}}`格式的内容（如`{{文章标题}}`、`{{版本来源}}`、`{{说明差异}}`）必须替换为实际内容并保留在最终文章中
  - **操作指南**：以`**`开头的规则说明（如`**正文穿插Emoji规范:**`、`**开头Emoji组合:**`等）是给创建者看的操作说明，**不得出现在最终正文中**
- **禁止复制规则说明**：模板中任何以列表形式出现的操作规范、规则说明都不应复制到最终文章中
- **仅保留内容部分**：最终文章只应包含实际的文章内容，不应包含任何操作指南或规则说明

### �📂 输出文件
- `d:\workspace\{级别}美文\{序号}.{作品名}-{作者}.md`

### ✅ 完成确认

## 📋 Step 5️⃣: 去重检查 (prose-deduplicator)

### 🎯 执行要求
- **必须调用skill**：`prose-deduplicator/SKILL.md`
- **总索引预检**：收录前必须查看`美文总索引.md`，确认重复情况
- **同一文件夹规则**：绝对不允许同一文件夹内出现相同文章
- **跨文件夹数量限制**：同一作品（作品名+作者）在全库中不得超过2篇，包含2篇
- **超限处理**：如发现某作品已达2篇上限，必须选择其他作品替代，不得强行收录
- **四级去重检查**：
  1. 总索引核查
  2. 文件名去重
  3. 作品去重
  4. 内容去重

### ✅ 检查清单
#### 收录前检查
- [ ] 已查看《美文总索引》，确认拟收录作品的现有情况
- [ ] 确认该作品全库收录数量未超过2篇限制
- [ ] 如需重复收录，已准备建立双向链接

#### 文件夹内检查  
- [ ] 序号无重复（1-100）
- [ ] 作品名+作者在同一文件夹内无重复
- [ ] 片段与完整文章一致

#### 跨文件夹检查
- [ ] 跨文件夹相同文章已建立双向链接
- [ ] 双向链接格式正确
- [ ] 《美文总索引》已更新重复关系记录

### 🚨 违规处理
- [ ] 发现超限：立即停止收录，选择替代作品
- [ ] 遗漏链接：补建所有跨文件夹重复文章的双向链接
- [ ] 索引不符：更新`美文总索引.md`，确保信息准确
- [ ] 未跳过任何skill要求

## 📋 Step 6️⃣: 建立链接 (prose-link-weaver)

### 🎯 执行要求
- **必须调用skill**：`prose-link-weaver/SKILL.md`
- **链接类型**：
  1. **片段→完整文章**：在片段下方添加`[[完整文章的文件名]]`
  2. **完整文章→片段**：在文章中添加`[[片段收集文件名]]`
  3. **文章→文章**：在相关作品中添加同主题文章的链接
  4. **索引链接**：更新索引文件的所有链接
- **链接格式**：使用标准Obsidian链接格式`[[文件名]]`
- **链接规范**：仅使用文件名，不包含路径，确保可点击跳转

### 📂 需要更新的文件
- `{级别}美文精选100篇.md`
- 所有完整文章的`.md`文件
- `美文收集索引.md`
