## 🚀 快速开始

### 在 Obsidian 中使用

1. **创建新仓库**
   - 打开 Obsidian
   - 点击 "Create new vault"
   - 选择 `openclaw-tutorial` 文件夹

2. **启用必要插件**
   - 设置 → 核心插件 → 启用以下插件：
     - ✅ 页面预览（用于悬浮预览双向链接）
     - ✅ 关系图谱（可视化笔记关系）
     - ✅ 模板（可选，用于创建新笔记）

3. **开始使用**
   - 打开 `OpenClaw-Complete-Tutorial.md`
   - 点击目录中的双向链接跳转到对应章节
   - 图片会自动嵌入显示

## ❓ 常见问题

### Q: 教程中的图片无法显示？

**A**: 确保：
1. 图片文件在 `images/` 目录下
2. Markdown 文件和图片在同一目录层级
3. 使用相对路径 `images/filename.png`

### Q: 双向链接在 Obsidian 中不起作用？

**A**: 确保：
1. 启用了「页面预览」核心插件
2. 链接格式正确：`[[#标题名称]]`
3. 标题名称完全匹配（包括大小写）

### Q: 可以在其他笔记软件中使用吗？

**A**: 可以！本教程使用标准 Markdown：
- 双向链接可能不被支持，但会显示为普通文本
- 图片使用标准 Markdown 语法，兼容性良好
- 代码块、表格等格式通用
