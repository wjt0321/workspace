# SecureVault 密码管理器设计文档

## 项目概述

### 项目名称
SecureVault - 类似1Password的企业级密码管理系统

### 项目目标
- 提供企业级密码安全管理解决方案
- 实现完整的Docker化部署
- 保持现代化UI/UX体验（与1Password一致）
- 确保端到端加密安全性
- 支持本地开发和Docker环境无缝切换

## 技术架构

### 前端技术栈
- **React 18**: 现代声明式UI框架
- **TypeScript**: 类型安全的开发体验
- **Next.js 14**: React全栈框架，支持SSR
- **shadcn/ui**: 可定制化组件库
- **Tailwind CSS**: 实用优先的CSS框架

### 后端技术栈
- **Rust + Actix-web**: 高性能异步Web框架
- **PostgreSQL**: 主数据库
- **Redis**: 缓存和会话存储
- **RabbitMQ**: 异步事件处理

### 基础设施
- **Docker + Docker Compose**: 容器化部署
- **Nginx**: 反向代理
- **Let's Encrypt**: SSL证书管理

## 安全架构

### 加密方案
- **传输加密**: TLS 1.3
- **存储加密**: AES-256-GCM
- **密钥派生**: Argon2id算法
- **端到端加密**: libsodium Sealed Boxes

### 密钥管理
1. **主密钥**: 从用户主密码派生
2. **保险库密钥**: 每个保险库独立
3. **项目密钥**: 每个密码项目独立
4. **临时密钥**: 会话级别的临时密钥
