# Kimi Code 使用规则

> 当前默认 Kimi CLI 入口与使用原则。

## 当前状态
- 默认 `kimi` 已切到 npm 全局安装版本
- 当前默认版本：**0.6.0**
- 自动 `/login` 刷新链路已验证可用

## 使用原则
- CLI 直调是当前最稳定的使用方式
- 搜索、轻量任务、看图、多模态优先走 Kimi Code
- ACP 不通时，直接切回 CLI，不纠结协议层

## 命令面注意事项
- 新版以 `kimi -p "..."` 为主
- 旧 `--print` 已不再作为默认写法
- token 过期时，优先自动尝试：`echo /login | kimi`

## 维护规则
- 升级优先走 npm 全局安装
- 升级后必须验证：版本、默认入口、登录态、常用脚本
- 不要让旧独立版继续干扰默认 `kimi`
