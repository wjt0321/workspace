# AI记忆

这是小黑在 Obsidian 中维护的**第二脑**。

## 定位
- 用于沉淀可复用、可主题化、可链接的长期知识
- 利用 Obsidian 的双向链接、结构化笔记与 Git 同步增强检索与维护能力
- **不替代** 小黑工作区内的主记忆体系：`memory/YYYY-MM-DD.md` 与 `MEMORY.md`

## 隐私边界
- 可公开/可同步到远程仓库的主题知识：可以写入这里
- 隐私、敏感、密钥、账户、私人细节：**禁止** 写入这里
- 如需建立关联，只允许写指针：例如“详见小黑工作区某文件/某日期笔记”，不复制敏感正文

## 结构
- `00-中枢/`：总入口、当前工作台
- `01-每日记录/`：后续如需接管 daily note，可逐步接入
- `02-长期记忆/`：稳定偏好、长期规则、重要决策
- `03-工具经验/`：Kimi / WebBridge / OpenWond / 飞书 等工具经验
- `05-问题排查/`：故障模式、排查套路、已验证经验

## 工作原则
1. 原始流水在工作区 daily notes 保留
2. 高价值经验提炼为主题卡片
3. 小步迭代，不一开始做大迁移
4. 保证内容适合 Git 同步与远程查看

> 当前为最小可用版 v1，先跑通结构与导航，再决定是否扩容。
