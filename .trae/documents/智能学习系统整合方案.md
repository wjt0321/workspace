# 智能学习系统实施方案（优化版）

## 核心架构
- **Obsidian** 作为学习中枢（笔记、词库、复习系统）
- **n8n** 作为自动化引擎（连接所有工具，可自建部署）
- **NAS** 作为数据存储中心
- **LLM API** 作为智能助手（国产大模型 + Gemini混合使用）
- **Telegram** 作为消息推送渠道
- **Google NotebookLM** 作为学习材料整理与音频生成工具

## API选择方案

### LLM API选择（混合策略）

#### 🌟 高级任务（使用Gemini）
- 复杂推理、长文分析、多模态理解
- 学习材料总结与批改
- 考试真题深度解析

#### 💡 日常任务（使用国产模型）
1. **智谱AI (GLM-4)** - 性价比高，中文能力强
2. **DeepSeek** - 代码能力突出，价格实惠
3. **通义千问 (Qwen)** - 阿里出品，免费额度大
4. **Kimi (Moonshot)** - 长文本处理强

## 主要功能模块

### 1. 每日学习计划推送系统
- n8n定时触发（每天8:00）
- **Gemini/智谱AI** 生成个性化学习计划
- **飞书日历/滴答清单** 创建日程提醒
- **Telegram Bot** 推送学习计划和提醒
- Obsidian创建当日学习笔记

### 2. 智能词库自动更新系统
- 每周自动生成新词汇
- 保存到Obsidian词库文件夹
- 使用Flashcards和Spaced Repetition插件
- 同步到NAS备份

### 3. 学习状况追踪与曲线复习系统
- 记录每日学习数据到Obsidian
- n8n每周分析学习进度
- 实现间隔重复算法（SuperMemo SM-2）
- **飞书多维表格/Grafana** 可视化学习曲线

### 4. 智能题库搜索与定制系统
- n8n爬取免费题库资源
- **国产LLM** 筛选适合题目
- 根据学习水平定制练习
- 保存到Obsidian题库

### 5. AI智能学习助手
- **Coze（扣子）** 或 **Dify** 构建智能助手
- 24小时学习问答（接入Gemini/国产LLM）
- 学习进度分析
- 学习建议推送
- **Telegram Bot** 学习打卡提醒

### 6. 学习材料整理系统（NotebookLM）
- **Google NotebookLM** 导入学习资料
- AI自动生成学习摘要
- 生成**音频播客**（通勤/运动时听）
- 智能问答，深入理解材料
- 导出大纲到Obsidian

### 7. 语音学习系统
- **讯飞TTS** 生成单词音频（免费额度10万次/月）
- **讯飞STT** 练习发音评测
- Obsidian录制学习笔记
- 备选：Edge TTS（完全免费，通过n8n调用）

### 8. 数据备份与同步
- Obsidian → NAS实时同步
- **坚果云** 每周增量备份（免费版1GB/月流量）
- 历史版本保留30天
- 本地Git版本控制（推荐）

### 9. 学习成果可视化仪表板
- **Grafana** 自建仪表盘（NAS部署，完全免费）
- 学习时长热力图
- 词汇掌握度雷达图
- 考试成绩趋势
- 备选：飞书多维表格仪表盘

## n8n节点配置要点
```yaml
# Gemini API调用（复杂任务）
HTTP Request:
  URL: https://generativelanguage.googleapis.com/v1beta/models/gemini-pro:generateContent
  Headers:
    Content-Type: application/json
  Query:
    key: YOUR_GEMINI_API_KEY
  Body:
    contents:
      - parts:
          - text: "生成今日学习计划..."

# 国产LLM调用示例（智谱AI，日常任务）
HTTP Request:
  URL: https://open.bigmodel.cn/api/paas/v4/chat/completions
  Headers:
    Authorization: Bearer YOUR_API_KEY
  Body:
    model: glm-4-flash  # 免费高速版
    messages: [...]

# Telegram Bot推送
HTTP Request:
  URL: https://api.telegram.org/bot<YOUR_BOT_TOKEN>/sendMessage
  Method: POST
  Body:
    chat_id: YOUR_CHAT_ID
    text: |
      📚 今日学习计划
      {{ $json.content }}
    parse_mode: Markdown

# Edge TTS调用（完全免费）
Execute Command:
  command: edge-tts --text "Hello World" --write-media output.mp3
```
