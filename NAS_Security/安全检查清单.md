# 飞牛NAS安全检查清单

> [!danger] 紧急提醒
> 如果你的NAS已经暴露在公网，请立即按照此清单进行安全检查！

---

## 立即执行（5分钟内）

### 1. 切断公网访问

- [x] 登录路由器，删除所有指向NAS的端口转发规则 ✅ 2026-02-10
- [x] 登录飞牛NAS，关闭"远程访问"功能 ✅ 2026-02-10
- [x] 关闭"中继服务" ✅ 2026-02-10
- [x] 如有DDNS，暂停服务 ✅ 2026-02-10

### 2. 运行官方查杀脚本

```bash
# SSH登录NAS执行
curl -L https://static2.fnnas.com/aptfix/trim-sec -o trim-sec && chmod +x trim-sec && ./trim-sec
```

- [x] 脚本执行完成 ✅ 2026-02-10
- [x] 无异常报告 ✅ 2026-02-10

---

## 系统检查（30分钟内）

### 3. 检查异常进程

```bash
# 查看可疑进程
ps aux | grep -E "(gots|trim_https|unknown|miner|xmr)"

# 查看CPU占用高的进程
top -bn1 | head -20
```

- [x] 无陌生进程 ✅ 2026-02-10
- [x] CPU占用正常 ✅ 2026-02-10

### 4. 检查网络连接

```bash
# 查看异常连接
netstat -tulpn | grep ESTABLISHED
ss -tulpn | grep ESTABLISHED

# 查看监听端口
netstat -tulpn | grep LISTEN
```

- [x] 无异常外连 ✅ 2026-02-10
- [x] 端口监听正常 ✅ 2026-02-10

### 5. 检查系统文件

```bash
# 检查关键目录
ls -la /usr/sbin/ | grep -E "(gots|trim)"
ls -la /usr/trim/bin/ 2>/dev/null

# 检查计划任务
crontab -l
cat /etc/crontab
ls -la /etc/cron.d/
```

- [ ] 无异常文件
- [ ] 计划任务正常

### 6. 检查系统日志

```bash
# 查看系统日志
tail -n 100 /var/log/syslog
tail -n 100 /var/log/messages

# 查看登录日志
last
lastb
```

- [ ] 无异常登录记录
- [ ] 无异常操作记录

