## 第二部分：Python环境安装指南（2026最新版）

### 1. Python版本选择建议
目前Python的最新版本是3.14.x系列，但对于零基础学员，我推荐安装**Python 3.12.x**，原因如下：
- **稳定性高**：3.12是长期支持版本，经过充分测试
- **兼容性好**：大多数第三方库都已完美支持
- **资料丰富**：学习资料和解决方案最多

当然，你也可以选择3.13.x或3.14.x，但需要注意部分库可能还未完全兼容。

### 2. Windows系统安装详解（以Python 3.12.7为例）

#### 步骤1：下载安装包
1. 访问Python官网：https://www.python.org/downloads/
2. 点击黄色的“Download Python 3.12.7”按钮（或选择最新3.12.x版本）
3. 根据系统位数选择：
   - 64位系统：Windows installer (64-bit)
   - 32位系统：Windows installer (32-bit)

#### 步骤2：运行安装程序
```bash
# 以下为安装过程中的关键步骤说明，不是代码
# 1. 双击下载的安装包（如python-3.12.7-amd64.exe）
# 2. 勾选"Add Python 3.12 to PATH"（这是最重要的一步！）
# 3. 点击"Install Now"开始安装
# 4. 安装完成后，点击"Close"关闭安装程序
```

**关键提醒**：
- 务必勾选“Add Python 3.12 to PATH”，否则需要在命令行中手动配置环境变量
- 如果安装路径有中文或空格，可能导致后续问题，建议使用英文路径

#### 步骤3：验证安装是否成功
打开命令行工具（Windows按Win+R，输入cmd按回车），输入以下命令：

```bash
python --version
```

如果显示`Python 3.12.x`，说明安装成功！

#### 步骤4：安装代码编辑器（推荐VS Code）
对于初学者，推荐使用Visual Studio Code（VS Code），它免费、轻量且功能强大。

**VS Code安装步骤**：
1. 访问：https://code.visualstudio.com/
2. 下载对应系统的安装包
3. 安装完成后，打开VS Code
4. 安装Python扩展：
   - 点击左侧扩展图标（四个方块）
   - 搜索"Python"
   - 点击安装Microsoft官方提供的Python扩展

### 3. macOS系统安装详解

## 第三部分：第一个Python程序讲解（详细注释版）

现在让我们开始编写第一个Python程序。我会为每行代码添加详细的中文注释，确保你理解每一部分的作用。

### 1. Hello World：最基本的打印语句

```python
# print()函数是Python中最常用的输出函数
# 括号内的内容可以是字符串（用引号包裹）、数字、变量等
print("Hello, Python!")  # 打印字符串"Hello, Python!"到屏幕上

# 打印多个内容，用逗号分隔
# Python会自动在逗号分隔的内容之间添加空格
print("我的名字是：", "学习搭子")  # 输出：我的名字是： 学习搭子

# 打印数字
print(2026)  # 直接打印数字2026，不需要引号

# 混合打印字符串和数字
print("今年是", 2026, "年")  # 输出：今年是 2026 年
```

### 2. 使用变量存储数据

```python
# 变量就像是一个标签，可以贴在数据上方便使用
# 变量名由字母、数字和下划线组成，不能以数字开头

# 定义变量并赋值
name = "小明"  # 将字符串"小明"存储到变量name中
age = 25       # 将数字25存储到变量age中

# 使用变量
print("姓名：", name)  # 输出：姓名： 小明
print("年龄：", age)   # 输出：年龄： 25

# 修改变量的值
age = 26  # 将age变量的值更新为26（覆盖原来的25）
print("明年年龄：", age)  # 输出：明年年龄： 26
```

### 3. 用户输入互动

```python
# input()函数可以让用户输入内容，程序会等待用户输入
# input()总是返回字符串类型，即使输入的是数字

# 获取用户输入
user_name = input("请输入你的名字：")  # 显示提示信息，等待用户输入
print("你好，", user_name, "！欢迎学习Python！")  # 使用用户输入的内容

# 输入数字并计算
num_str = input("请输入一个数字：")  # input()返回的是字符串
num = int(num_str)  # 用int()函数将字符串转换为整数
print("你输入的数字是：", num)
print("这个数字加10等于：", num + 10)  # 可以进行数学运算

# 更简洁的写法（一行完成输入和转换）
num2 = int(input("请输入第二个数字："))
print("第二个数字的平方是：", num2 * num2)
```

## 第五部分：常见问题解答

### Q1：安装时忘记勾选"Add Python to PATH"怎么办？
A：有两种解决方法：
1. **重新运行安装程序**：选择"Modify" → 勾选"Add Python to PATH" → 完成修改
2. **手动配置环境变量**：
   - 找到Python安装路径（如`C:\Users\用户名\AppData\Local\Programs\Python\Python312`）
   - 将此路径添加到系统环境变量PATH中

### Q2：命令行输入python没反应？
A：可能的原因和解决方法：
1. **需要重启终端**：安装完成后，需要关闭并重新打开命令行工具
2. **使用python3命令**：某些系统需要使用`python3`命令
3. **检查安装路径**：确认Python已正确安装

### Q3：代码报错了怎么办？
A：这是学习编程的正常过程！请：
1. **仔细阅读错误信息**：Python的错误信息很友好，会告诉你错误类型和位置
2. **常见错误**：
   - 拼写错误：print写成了prin
   - 语法错误：忘记加冒号、括号不匹配
   - 缩进错误：Python使用缩进表示代码块，必须保持一致
3. **尝试自己解决**：这是最好的学习方式
