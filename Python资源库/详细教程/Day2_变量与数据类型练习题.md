## 一、知识回顾与最新视频推荐

### 1.1 变量基础概念回顾

**变量**就像是贴了标签的储物盒，用来存储程序运行时的数据：
- **变量名** = 盒子的标签
- **变量值** = 盒子里的东西
- **内存空间** = 盒子本身

**变量使用三步曲**：
1. **定义变量**：创建储物盒并贴上标签
2. **赋值**：往盒子里放东西
3. **使用/修改**：查看或更换盒子里的东西

### 1.2 Python主要数据类型介绍

Python有四种最基础的数据类型，你目前只需要掌握前三种：

| 数据类型 | 英文名 | 示例 | 用途说明 |
|---------|--------|------|---------|
| **整数** | int | `age = 25` | 表示没有小数部分的数字，用于计数、年龄等 |
| **浮点数** | float | `price = 19.9` | 表示带小数点的数字，用于价格、测量值等 |
| **字符串** | str | `name = "小明"` | 表示文本信息，用引号包裹 |
| **布尔值** | bool | `is_student = True` | 表示真(True)或假(False)，用于条件判断 |

### 1.3 类型查看神器：type()函数

使用`type()`函数可以查看任何数据或变量的类型：

```python
# 示例1：直接查看数据的类型
print(type(25))        # 输出：<class 'int'>      → 25是整数类型
print(type(19.9))      # 输出：<class 'float'>    → 19.9是浮点数类型
print(type("小明"))    # 输出：<class 'str'>      → "小明"是字符串类型
print(type(True))      # 输出：<class 'bool'>     → True是布尔值类型

# 示例2：查看变量的类型
age = 25               # 定义一个变量age，存储整数25
age_type = type(age)   # 使用type()查看age的类型，结果存储在age_type中
print(age_type)        # 输出：<class 'int'>      → 变量age存储的是整数

# 示例3：变量本身没有类型，只有存储的数据有类型
name = "小明"          # name存储字符串
print(type(name))      # 输出：<class 'str'>      → 查看的是name存储的数据类型

name = 25              # 同一个name变量现在存储整数
print(type(name))      # 输出：<class 'int'>      → 类型随存储的数据而变化
```

**重要理解**：变量本身没有固定类型，就像一个空盒子，今天可以装苹果（字符串），明天可以装鸡蛋（整数）。`type()`查看的是盒子当前装的东西的类型。

### 题目2：数据类型转换（掌握转换规则）

**要求**：
1. 将字符串"100"转换为整数
2. 将整数100转换为浮点数
3. 将浮点数3.14转换为整数（注意观察小数部分）
4. 将整数25转换为字符串，然后与其他字符串拼接

```python
# ====== 题目2：你的代码写在这里 ======

# 1. 字符串转整数
str_num = "100"                 # 定义一个字符串变量，内容是"100"
int_num = int(str_num)         # 使用int()函数将字符串转换为整数
print("字符串'100'转整数：", int_num, "，类型：", type(int_num))

# 2. 整数转浮点数
original_int = 100             # 定义一个整数变量
float_num = float(original_int) # 使用float()函数将整数转换为浮点数
print("整数100转浮点数：", float_num, "，类型：", type(float_num))

# 3. 浮点数转整数（注意小数部分）
pi = 3.14                      # 定义一个浮点数变量
int_pi = int(pi)               # 使用int()函数将浮点数转换为整数
print("浮点数3.14转整数：", int_pi, "，类型：", type(int_pi))
print("注意：转换后小数部分.14被丢弃了，只保留整数部分3")

# 4. 整数转字符串并拼接
age = 25                       # 定义一个整数变量
str_age = str(age)             # 使用str()函数将整数转换为字符串
message = "我今年" + str_age + "岁"  # 字符串拼接
print("拼接结果：", message)
```

**重要规则总结**：
1. **字符串转数字**：字符串内容必须是纯数字，如"100"可以，"100元"不行
2. **浮点数转整数**：直接丢弃小数部分，不进行四舍五入
3. **任何类型转字符串**：都可以用`str()`，不会出错

### 题目3：字符串基本操作（文本处理入门）

**要求**：
1. 创建两个字符串变量，分别存储你的姓氏和名字
2. 使用`+`拼接成完整姓名
3. 使用字符串切片获取姓氏的第一个字
4. 使用`len()`函数获取完整姓名的长度
5. 尝试使用字符串的`.upper()`方法将姓名转为大写

```python
# ====== 题目3：你的代码写在这里 ======

# 1. 定义姓氏和名字
last_name = "张"               # 存储姓氏
first_name = "三"              # 存储名字

# 2. 字符串拼接
full_name = last_name + first_name  # 使用+号连接两个字符串
print("完整姓名：", full_name)

# 3. 字符串切片（获取第一个字符）
# 字符串索引从0开始，last_name[0]表示第一个字符
first_char = last_name[0]      # 获取姓氏的第一个字
print("姓氏第一个字：", first_char)

# 4. 获取字符串长度
name_length = len(full_name)   # len()函数返回字符串的字符数量
print("姓名长度：", name_length, "个字符")

# 5. 转换为大写
upper_name = full_name.upper() # .upper()方法将字符串所有字母转为大写
print("大写姓名：", upper_name)

# 扩展思考：如果是英文名，比如"Tom"，.upper()会有什么效果？
```

**字符串操作小技巧**：
- **索引**：`字符串[位置]`，位置从0开始计数
- **切片**：`字符串[开始:结束]`，获取从开始到结束（不含）的子串
- **长度**：`len(字符串)`，返回字符数量（中文、英文都算1个）
- **方法**：`字符串.方法名()`，如`.upper()`、`.lower()`、`.strip()`

### 题目4：综合应用——简易个人信息卡

**要求**：
1. 使用`input()`函数让用户输入年龄（整数）、身高（浮点数）、姓名（字符串）
2. 注意：`input()`默认返回字符串，需要转换为正确的类型
3. 计算BMI指数（体重假设为60公斤，BMI = 体重/身高²）
4. 输出格式美观的个人信息卡

```python
# ====== 题目4：你的代码写在这里 ======

# 1. 使用input()获取用户输入（注意：input()返回的都是字符串）
print("=== 请填写个人信息 ===")
input_name = input("请输入你的姓名： ")           # 获取姓名，已经是字符串
input_age_str = input("请输入你的年龄： ")       # 获取年龄，是字符串"25"
input_height_str = input("请输入你的身高(米)： ") # 获取身高，是字符串"1.75"

# 2. 类型转换（重要！）
# input_age_str是字符串，如"25"，需要转换为整数才能做数学计算
input_age = int(input_age_str)                  # 将字符串"25"转换为整数25
input_height = float(input_height_str)          # 将字符串"1.75"转换为浮点数1.75

# 3. 计算BMI（假设体重为60公斤）
weight = 60                                      # 固定体重60公斤
bmi = weight / (input_height ** 2)              # BMI公式：体重/身高²
                                                # input_height ** 2 表示身高的平方

# 4. 输出个人信息卡
print("\n" + "="*40)
print("           个人信息卡")
print("="*40)
print(f"姓名：{input_name}")                    # f-string格式化，更简洁
print(f"年龄：{input_age}岁")
print(f"身高：{input_height}米")
print(f"BMI指数：{bmi:.2f}")                    # :.2f表示保留2位小数

# 5. BMI健康判断
if bmi < 18.5:
    status = "偏瘦"
elif bmi < 24:
    status = "正常"
elif bmi < 28:
    status = "偏胖"
else:
    status = "肥胖"
    
print(f"健康状况：{status}")
print("="*40)
```

**关键知识点**：
1. `input()`函数：总是返回字符串，即使输入的是数字
2. 类型转换的必要性：字符串"25"不能做数学计算，需要`int("25")`转为整数25
3. `f-string`格式化：`f"姓名：{name}"`比`"姓名：" + name`更简洁
4. 条件判断`if-elif-else`：根据BMI值判断健康状态

### 题目5：布尔值应用（逻辑思维训练）

**要求**：
1. 创建两个布尔变量，表示是否为学生、是否有工作
2. 使用逻辑运算符`and`、`or`、`not`进行组合判断
3. 设计一个场景：判断某人是否可以享受学生优惠

```python
# ====== 题目5：你的代码写在这里 ======

# 1. 定义布尔变量
is_student = True              # 是否是学生
has_job = False                # 是否有工作

print("是否是学生：", is_student)
print("是否有工作：", has_job)

# 2. 逻辑运算符演示
# and：两个条件都为True时，结果为True
can_get_discount = is_student and (not has_job)  # 是学生且没有工作
print("是否可以享受学生优惠：", can_get_discount)

# or：至少一个条件为True时，结果为True
need_financial_aid = (not has_job) or is_student  # 没有工作或是学生
print("是否需要经济援助：", need_financial_aid)

# not：取反，True变False，False变True
is_working_adult = not is_student and has_job    # 不是学生且有工作
print("是否是在职成人：", is_working_adult)

# 3. 实际应用场景
age = 20
is_full_time_student = True
has_student_card = True

# 学生优惠条件：年龄≤22岁，且是全职学生，且有学生证
eligible_for_discount = (age <= 22) and is_full_time_student and has_student_card
print(f"\n实际场景判断：")
print(f"年龄：{age}岁，全职学生：{is_full_time_student}，有学生证：{has_student_card}")
print(f"符合学生优惠条件：{eligible_for_discount}")
```

**布尔逻辑要点**：
- `and`：严格，所有条件都要满足
- `or`：宽松，只要一个条件满足
- `not`：反转，真变假，假变真
- 应用：条件判断、权限控制、业务规则等
