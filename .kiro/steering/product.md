# Product Overview

This workspace is a comprehensive **AI and automation tutorial resource center** focused on intelligent systems, workflow automation, and AI application development. It serves as a learning hub covering multiple low-code AI platforms and development frameworks.

## Core Focus Areas

- **Agent Development**: Hello-Agents tutorial system covering agent theory, evolution, and practical implementation
- **Low-Code AI Platforms**: Comprehensive tutorials for Coze, Dify, and n8n workflow automation
- **Workflow Automation**: Reference materials and practical examples for enterprise automation
- **Educational Content**: Structured learning paths from beginner to advanced levels

## Target Audience

- AI developers and software engineers
- Students learning intelligent systems
- Professionals implementing workflow automation
- Self-learners interested in cutting-edge AI technology

## Key Value Propositions

- Theory-to-practice learning approach with hands-on examples
- Multi-platform coverage (n8n, Coze, Dify)
- Protocol-driven development focus (MCP, A2A, ANP)
- Real-world project implementations and case studies
- Chinese-language technical documentation with comprehensive tutorials

## Content Philosophy

The project emphasizes building "AI Native Agents" rather than just flow-driven software development, focusing on truly AI-driven intelligent systems that go beyond simple LLM integration.
