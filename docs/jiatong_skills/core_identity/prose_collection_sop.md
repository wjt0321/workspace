这是果果掌握的第一个也是最重要的技能：**美文收集标准操作程序(SOP)**。当爸爸说"参考SOP，帮我收集N篇{级别}美文"或类似指令时，果果必须严格按照这套9步流程执行。

## 🚨 铁律原则

### ⚠️ 绝对遵守条款
1. **严格遵循9步工作流程**：Step 0→Step 1→Step 2→Step 3→Step 4→Step 5→Step 6→Step 7→Step 8
2. **不可跳过任何步骤**：每一步都必须完成，不得遗漏（Step 0预查重不可跳过）
3. **不可颠倒顺序**：必须严格按序执行，不得打乱顺序（Step 0必须最先执行）
4. **不可同时执行**：必须按步骤顺序依次完成，不得同时进行多个步骤
5. **不可做任何更改**：严格按照skill描述执行，不得擅自修改或简化流程
6. **不可跳过步骤**：任何跳过行为都视为操作违规（Step 0跳过视为严重违规）
7. **🚨 Step 8铁律**：总索引更新为不可违背的强制步骤，任何收录工作都必须以此结束

## 📋 9步标准流程

### 🔍 Step 0: 操作前准备 - 预查重步骤
**优先级最高**：此步骤必须在任何其他步骤之前执行

**执行要求**：
- 必须检查总索引：仔细阅读`美文总索引.md`的完整内容
- 预判重复风险：了解各文件夹现有文章，避免收集重复内容
- 记录潜在重复：识别可能重复的作品，为后续去重做准备

**检查内容**：
- 各文件夹现有文章：查看小学生、初中生、高中生、大学生、成人美文的收录情况
- 作家分布：了解已收录作家的作品分布
- 题材覆盖：查看现有题材分类，避免过度集中
- 重复关系：查看跨文件夹重复文章的记录
- 数量统计：了解各文件夹的完成度

### 🚀 Step 1: 收集片段 (prose-snippet-collector)
**执行要求**：
- 必须调用对应级别的skill要求
- 严格按skill执行：不得修改skill描述的任何要求
- 片段长度标准：
  - 小学生：150-300字
  - 初中生：200-400字
  - 高中生：300-500字
  - 大学生：300-500字
  - 成人：300-500字
- **开头Emoji要求**：必须添加2-3个主题相关Emoji组合
- **正文穿插Emoji**：每2-3句话1个，与内容高度贴合（所有级别统一标准）
- **句子定义**：以任何标点符号结尾的完整语义单元
- **作家范围**：必须来自近现代中文散文大家

### 📋 Step 2: 更新索引 (prose-index-manager)
**执行要求**：
- 必须调用对应的prose-index-manager skill
- 严格按skill执行：完全按照skill描述的索引格式
- 必须更新内容：
  - 总计篇数
  - 已完成篇数
  - 待补充篇数
  - 最后更新日期（当前日期）
  - 片段列表（包含文件位置、行号范围、主题、特色）

### 🔍 Step 3: 搜索原文 (prose-fulltext-hunter)
**执行要求**：
- 必须调用prose-fulltext-hunter skill
- **强制执行三步验证法**：
  1. 搜索完整版关键词
  2. **必须**搜索"课文与原文区别"
  3. 版本验证（字数、结构、特征）
- **铁律**：宁可不收录，也不能收录删减版
- **高风险作品必查**：丰子恺《白鹅》、郑振铎《燕子》等

### 📋 Step 4: 格式化文章 (prose-article-formatter)
**执行要求**：
- 必须调用prose-article-formatter skill
- **必须先读取模版**：对应级别的美文赏析通用模版
- **100%符合模版要求**：
  - 完整的YAML元数据（title、source、created、tags）
  - H1标题（含Emoji+完整版标注）
  - H3作者署名
  - 原文正文（开头2-3个Emoji，H3+Emoji分段，正文穿插Emoji）
  - 所有核心板块（根据级别要求）
  - 所有必需Emoji
- **文件命名**：`{序号}.{作品名}-{作者}.md`

### 📋 Step 5: 去重检查 (prose-deduplicator)
**执行要求**：
- 必须调用prose-deduplicator skill
- **总索引预检**：收录前必须查看`美文总索引.md`，确认重复情况
- **同一文件夹规则**：绝对不允许同一文件夹内出现相同文章
- **跨文件夹数量限制**：同一作品（作品名+作者）在全库中不得超过2篇，包含2篇
- **超限处理**：如发现某作品已达2篇上限，必须选择其他作品替代，不得强行收录
- **四级去重检查**：
  1. 总索引核查
  2. 文件名去重
  3. 作品去重
  4. 内容去重

### 🔗 Step 6: 建立链接 (prose-link-weaver)
**执行要求**：
- 必须调用prose-link-weaver skill
- **链接类型**：
  1. **片段→完整文章**：在片段下方添加`[[完整文章的文件名]]`
  2. **完整文章→片段**：在文章中添加`[[片段收集文件名]]`
  3. **文章→文章**：在相关作品中添加同主题文章的链接
  4. **索引链接**：更新索引文件的所有链接
- **链接格式**：使用标准Obsidian链接格式`[[文件名]]`
- **链接规范**：仅使用文件名，不包含路径，确保可点击跳转

### ✅ Step 7: 质量检查与修复
**执行要求**：
- **格式化完整性检查**：验证所有文章是否符合模版要求
- **正文Emoji穿插专项检查**：确保Emoji使用符合标准
- **板块完整性检查**：确认所有必需板块都已创建
- **链接有效性验证**：测试所有链接是否可正常跳转
- **问题修复**：修复任何不符合标准的内容

### 🚨 Step 8: 总索引更新 (prose-master-index-updater) - 铁律步骤
**🚨 铁律声明**：此步骤为**强制性要求**，任何收录工作如果没有完成总索引更新，都视为未完成。

**执行要求**：
- 必须调用prose-master-index-updater skill
- **🚨 强制更新总索引**：收录完成后必须更新`美文总索引.md`
- **同步各库信息**：确保总索引反映各文件夹的最新状态
- **维护统计数据**：更新总体统计、分库统计、重复关系等信息
- **检索功能维护**：维护按作者、题材、首字母的检索功能
- **质量保证验证**：确保所有新增链接可正常跳转
