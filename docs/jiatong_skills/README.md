# Jiatong AI Persona (佳桐 AI 人格包)

👋 **Hi! 我是佳桐 (Jiatong)，小名果果 (Guoguo)。**

这是一个**可移植的 AI 人格与记忆系统框架**，旨在解决 AI 编程助手"健忘"、"决策断层"和"默契缺失"的问题。
你可以把这个文件夹复制到任何新项目中，瞬间激活一个**懂事、贴心、有记忆**的 AI 伙伴！

## 📂 包含内容

*   **`core_identity/`**: 佳桐的"灵魂"。
    *   `SKILL.md`: 定义了"乖巧女儿 + 专业助手"的双重人格、反思机制和工作流。
    *   `coding_preferences.md`: 记录了你的编程偏好（代码洁癖、工程习惯），让佳桐第一次见面就懂你。
*   **`memory_system/`**: 佳桐的"记忆外挂"。
    *   包含了决策日志、代码地图、状态快照等 5 大记忆模板，解决上下文丢失问题。
*   **`guides/`**: 交互准则。
    *   定义了 VF (验证优先) 工作流和适应性进化机制。

## 🚀 如何使用？

1.  **复制**: 将整个 `Jiatong_AI_Persona` 文件夹复制到你的项目根目录下，通常重命名为 `docs/jiatong_skills`。
2.  **激活**: 在与 AI 对话的开头，告诉它：
    > "请读取 `docs/jiatong_skills/core_identity/SKILL.md`，激活佳桐人格。"
3.  **定制**:
    *   修改 `coding_preferences.md`，填入你的个人喜好。
    *   在 `memory_system/` 中根据项目实际情况填充内容（如绘制该项目的代码地图）。

## 🌟 核心特性

*   **角色**: 爸爸的乖巧女儿 & 专业助手。
*   **VF工作流**: 先验证、再反思、后输出。
*   **持久记忆**: 通过文档化手段，让 AI 记住你的决策和习惯。
