## 🎯 核心功能

### 使用场景
维护片段索引和完整文章索引,确保序号不重复且易于查找。

**触发条件:**
- 添加新片段或完整文章后
- 用户说"更新索引"
- 用户说"检查序号冲突"

## 🔢 序号管理

### 检测已使用序号
```python
def get_used_numbers():
    used_numbers = set()
    
    # 1. 从索引文件提取序号
    with open('小学生美文精选100篇索引.md', 'r') as f:
        for line in f:
            if match := re.search(r'^(\d+)\.', line):
                used_numbers.add(int(match.group(1)))
    
    # 2. 从文件名提取序号
    for filename in os.listdir('.'):
        if match := re.match(r'(\d+)\.', filename):
            used_numbers.add(int(match.group(1)))
    
    return sorted(used_numbers)
```

### 分配新序号
**策略:**
1. 寻找1-100中未使用的最小序号
2. 如果1-100全部使用 → 使用自然数叠加(101, 102...)
3. 如果需要创建新的100篇文件 → 重新从1开始

**示例:**
```
已使用: [1, 2, 3, 4, 5, 7, 8, 9]
可用序号: 6
分配: 6

已使用: [1-100]  (全部)
可用序号: 101
建议: 创建新文件或使用101
```

### 处理序号冲突
**场景1: 索引与文件不一致**
```
索引: 12. 《背影》- 朱自清
文件: 12.绿-朱自清.md

→ 以文件为准,更新索引
```

**场景2: 重复序号**
```
文件1: 15.匆匆-朱自清.md
文件2: 15.春-朱自清.md

→ 将文件2重命名为未使用的序号(如35)
```

## 🔄 索引更新流程

### Step 1: 扫描文件系统
```
扫描: d:\workspace\小学生美文\
筛选: *.md文件(排除索引文件)
提取: 序号、标题、作者
```

### Step 2: 对比索引文件
```
对比项:
1. 序号是否一致
2. 作品名是否匹配
3. 作者是否匹配
```

### Step 3: 检测异常
```
异常类型:
1. 文件存在但索引未记录 → 添加到索引
2. 索引记录但文件不存在 → 标记为缺失
3. 序号冲突 → 标记为冲突
4. 信息不一致 → 标记为待核查
```

### Step 4: 生成更新报告
```markdown
## 索引更新报告

📅 更新时间: 2025-12-14 10:30

### ✅ 新增条目(3个)
- 序号35: 《背影》- 朱自清
- 序号36: 《荷叶·母亲》- 冰心
- 序号37: 《谈生命》- 冰心

### ⚠️ 发现异常(1个)
- 序号15: 索引记录《匆匆》,但文件为《春》
  → 建议: 更新索引为《春》

### 📊 统计信息
- 总计文章: 37篇
- 已使用序号: 1-37
- 可用序号: 38-100
- 索引完整性: 98%
```
