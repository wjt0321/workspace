# 🔄 美文收集主工作流程 (prose-workflow-master)

## 📋 Skill 元数据
```yaml
name: prose-workflow-master
version: 3.0.0
description: 美文收集主工作流程控制器，协调9个步骤按标准顺序执行
author: AI Assistant
created: 2025-12-21
tags: [工作流程, 主控制器, 9步流程, 自动化]
```

## 🎯 核心功能

### 使用场景
当用户需要收集美文时，使用此主skill自动执行完整的9步工作流程。

**触发条件:**
- 用户说"参考.skill，帮我收集N篇{级别}美文"
- 用户说"按照skill流程收集美文"
- 用户说"完整收集{级别}美文"

## ⚠️ 核心原则

### 铁律1：严格按序执行
```
Step 0 → Step 1 → Step 2 → Step 3 → Step 4 → Step 5 → Step 6 → Step 7 → Step 8
```

**不允许的操作：**
- ❌ 跳过任何步骤
- ❌ **严重违规**：跳过Step 0预查重
- ❌ 颠倒执行顺序
- ❌ 只执行部分步骤
- ❌ 同时执行多个步骤
- 🚨 **特别注意**：Step 8（总索引更新）为铁律步骤，不可违背、不可跳过！

**正确的执行：**
- ✅ 严格按照0→1→2→3→4→5→6→7→8的顺序
- ✅ **优先执行Step 0**：预查重为最高优先级
- ✅ 每一步完成后才能进入下一步
- ✅ 全部9步完成才算结束
- 🚨 **Step 8必须执行**：任何收录工作都必须以更新总索引为结束

### 铁律2：基础四项检查
任何文件操作必须进行以下四项正确性检查：
1. **序号连续准确性**：确保所有文章和片段的序号连续无重复
2. **双向链接完整性**：
   - 片段→完整文章的链接
   - 完整文章→片段的链接
3. **索引正确性**：确保索引文件中的统计信息和链接准确无误
4. **文章清洗去重**：执行三级去重检查，相同文章清除后必须补足数量

**执行时机：**
- ✅ 每一步操作前后都需检查
- ✅ 最终完成前进行全面检查
- ✅ 去重可在过程中进行，但需确保数量补足

## 🏗️ 完整工作流程

### 🔍 Step 0️⃣: 操作前准备 - **预查重步骤（优先级最高）**

**执行内容：**
1. **必须先检查总索引**：读取并分析`美文总索引.md`的完整内容
2. **了解现有分布**：查看各文件夹的文章收录情况、作家分布、题材覆盖
3. **识别重复风险**：标记可能重复的作品和作家，识别潜在冲突
4. **规划收集策略**：制定非重复作品收集策略，避免低效重复工作

**输出文件：**
- 预查重分析报告（记录在执行日志中）
- 非重复作品收集策略清单

**检查点：**
- [ ] 已完整阅读并分析《美文总索引.md》
- [ ] 已了解各文件夹现有文章分布
- [ ] 已标记潜在重复风险作品
- [ ] 已规划非重复收集策略

**完成标志：** ✅ "预查重完成：已规划非重复策略"

**违规后果：** 🚨 跳过此步骤视为严重违规，必须从Step 0重新开始

---

### Step 1️⃣: 收集片段 (prose-snippet-collector) - **执行前必须完成Step 0**

**执行内容：**
1. 识别目标级别（小学/初中/高中/大学/成人）
2. 创建或打开片段收集文件：`{级别}美文精选100篇.md`
3. 为N篇文章收集经典片段（200-400字）
4. 每个片段包含：序号、作品名、作者、引用内容、推荐理由
5. 避免重复（检查已有片段）

**输出文件：**
- `d:\workspace\{级别}美文\{级别}美文精选100篇.md`

**检查点：**
- [ ] 片段文件已创建/更新
- [ ] N个片段已添加
- [ ] 每个片段格式正确（包含所有必需元素）
- [ ] 序号连续无重复

**完成标志：** ✅ "片段收集完成：N个片段"

---

### Step 2️⃣: 更新索引 (prose-index-manager)

**执行内容：**
1. 打开索引文件：`美文收集索引.md`
2. 在"片段列表"中添加新收集的片段信息
3. 更新统计数据：
   - 总计篇数
   - 片段收集数
   - 完整文章数（当前为待收集）
   - 最后更新日期
4. 保持索引结构完整

**输出文件：**
- `d:\workspace\{级别}美文\美文收集索引.md`（更新）

**检查点：**
- [ ] 索引文件已更新
- [ ] 片段列表已添加新项
- [ ] 统计数据正确
- [ ] 格式符合规范

**完成标志：** ✅ "索引更新完成"

---

### Step 3️⃣: 搜索原文 (prose-fulltext-hunter)

**执行内容：**
1. 对每个片段执行完整版原文搜索
2. **强制执行三步验证法：**
   - 第一步：搜索完整版关键词
   - 第二步：**必须**搜索"课文与原文区别"
   - 第三步：版本验证（字数、结构、特征）
3. 对高风险作品（如丰子恺《白鹅》）特别核查
4. 记录版本核查结果

**核查清单（每篇）：**
- [ ] 已搜索"课文与原文区别"
- [ ] 已确认是否存在删改
- [ ] 已对比字数
- [ ] 确认为完整版 ✅ 或 标记问题 ⚠️

**检查点：**
- [ ] 所有N篇文章已核查
- [ ] 确认均为完整版（或标记待核查）
- [ ] 记录了版本信息

**完成标志：** ✅ "原文搜索完成：N篇完整版已确认"

---

### Step 4️⃣: 格式化文章 (prose-article-formatter)

**执行内容：**
1. **必须先读取对应级别的模版文件**
   - 路径：`d:\workspace\{级别}美文\美文赏析与教学通用模版.md`
2. 为每篇文章按照模版创建完整文件
3. 文件命名：`{序号}.{作品名}-{作者}.md`
4. **100%符合模版要求**：
   - 完整的YAML元数据
   - 所有板块（初中7个、高中8个等）
   - 所有Emoji标记
   - 完整的原文或标注原文位置

**输出文件（每篇）：**
- `d:\workspace\{级别}美文\{序号}.{作品名}-{作者}.md`

**检查点（每篇）：**
- [ ] 文件已创建
- [ ] YAML元数据完整
- [ ] 标题含Emoji
- [ ] 所有板块齐全
- [ ] 格式符合模版

**完成标志：** ✅ "文章格式化完成：N篇文章已创建"

---

### Step 5️⃣: 去重检查 (prose-deduplicator)

**执行内容：**
1. 检查文件夹内所有`.md`文件
2. 执行三级去重检查：
   - **文件名去重**：检查序号是否重复
   - **作品去重**：检查"作品名+作者"组合是否重复
   - **内容去重**：检查是否有相同内容
3. 如发现重复，报告问题并建议处理方案

**检查清单：**
- [ ] 序号无重复（1-100范围内）
- [ ] 作品名+作者无重复
- [ ] 片段与完整文章一致
- [ ] 无内容重复

**检查点：**
- [ ] 去重检查已完成
- [ ] 无重复或重复已处理
- [ ] 所有文件序号连续

**完成标志：** ✅ "去重检查完成：无重复" 或 ⚠️ "发现重复，已处理"

---

### Step 6️⃣: 建立链接 (prose-link-weaver)

**执行内容：**
1. 在片段文件中为每个片段添加指向完整文章的链接
   - 格式：`[[{序号}.{作品名}-{作者}]]`
2. 在每篇完整文章中添加返回片段的链接
   - 在"相关作品"部分添加：`[[{级别}美文精选100篇#标题|返回片段]]`
3. 在完整文章间建立相关链接（同主题、同作者等）
4. 更新索引文件，添加所有链接

**需要更新的文件：**
- `{级别}美文精选100篇.md`（片段→文章链接）
- 所有完整文章的`.md`文件（文章→片段、文章→文章）
- `美文收集索引.md`（索引链接）

**检查点：**
- [ ] 片段→文章链接已建立（N个）
- [ ] 文章→片段链接已建立（N个）
- [ ] 文章→文章链接已建立（若干个）
- [ ] 索引文件链接已更新
- [ ] 所有链接可点击跳转

**完成标志：** ✅ "链接建立完成：X个双向链接"

---

### Step 7️⃣: 质量检查与修复

**执行内容：**
1. **格式化完整性检查**：验证所有文章是否符合模版要求
2. **正文Emoji穿插专项检查**：确保Emoji使用符合标准
3. **板块完整性检查**：确认所有必需板块都已创建
4. **链接有效性验证**：测试所有链接是否可正常跳转
5. **问题修复**：修复任何不符合标准的内容

**质量检查清单（每篇）：**
```markdown
## 格式化完整性检查清单

### 必需元素(不得缺项)
- [ ] YAML 元数据(title, source, created, tags)
- [ ] H1 标题(含Emoji +完整版标注)
- [ ] H3 作者署名
- [ ] 📖 原文正文(分段+Emoji)
- [ ] 🌟 美文赏析(3个子板块)
- [ ] 🎭 朗读指导(2个子板块)
- [ ] 🌈 小练笔(步骤式)
- [ ] 📚 知识小卡片

### 正文Emoji穿插专项检查
- [ ] 开头添加了2-3个主题相关Emoji组合
- [ ] 每个段落都有适度Emoji穿插
- [ ] Emoji密度符合年龄段要求
- [ ] 所有Emoji都与句子内容相关
- [ ] 没有影响阅读流畅性
```

**检查点：**
- [ ] 所有文章已通过质量检查
- [ ] 不符合标准的文章已修复
- [ ] Emoji使用符合年龄段标准
- [ ] 所有板块齐全无缺项
- [ ] 链接验证全部通过

**完成标志：** ✅ "质量检查完成：N篇文章符合标准"

---

### 🚨 Step 8️⃣: 总索引更新 (prose-master-index-updater) - **铁律步骤**

**执行内容：**
1. **🚨 强制更新总索引**：收录完成后必须更新`美文总索引.md`，这是不可违背的铁律
2. **同步各库信息**：确保总索引反映各文件夹的最新状态
3. **维护统计数据**：更新总体统计、分库统计、重复关系等信息
4. **检索功能维护**：维护按作者、题材、首字母的检索功能
5. **质量保证验证**：确保所有新增链接可正常跳转

> **⚠️ 铁律声明**：此步骤为强制性要求，任何收录工作如果没有完成总索引更新，都视为未完成。无论收录多少篇文章，都必须执行此步骤！

**总索引更新检查清单：**
```markdown
## 总索引更新检查清单

### 基础信息更新
- [ ] ✅ 总体统计数据已更新（总文章数、涉及作家数）
- [ ] ✅ 分库统计表已更新（各文件夹文章数量、完成度）
- [ ] ✅ 最后更新时间已修改为当前日期

### 文章信息同步
- [ ] ✅ 新收录文章已添加到对应文件夹分类中
- [ ] ✅ 按作者分类已更新（新作家或新作品）
- [ ] ✅ 按题材分类已更新（确保分类准确）
- [ ] ✅ 跨文件夹重复关系已记录（如有新增重复）

### 检索功能维护
- [ ] ✅ 按首字母检索已更新（新文章标题）
- [ ] ✅ 快速检索功能正常（链接有效性）
- [ ] ✅ 各库索引链接正常（指向各文件夹索引）

### 质量保证检查
- [ ] ✅ 所有新增链接可正常跳转
- [ ] ✅ 🔗标记正确标注重复文章
- [ ] ✅ 统计数据与实际文件一致
- [ ] ✅ 分类逻辑科学合理

### 🚨 铁律完成确认
- [ ] ✅ 强制更新已完成，不可违背的Step 8已执行
- [ ] ✅ 收录工作正式结束，总索引已同步最新状态
```

**输出文件：**
- `美文总索引.md`（根目录更新）

**完成标志：** 🚨 **"总索引更新完成：铁律步骤已执行，收录工作正式结束"**

## 🚨 错误处理

### 常见错误

#### 错误1：跳过步骤
```
❌ 错误：直接创建完整文章，跳过片段收集
✅ 修正：删除文章，从Step 1重新开始
```

#### 错误2：版本未验证
```
❌ 错误：使用了课文删减版
✅ 修正：重新执行Step 3，搜索完整版
```

#### 错误3：模版不符合
```
❌ 错误：文章缺少板块或Emoji
✅ 修正：重新执行Step 4，100%符合模版
```

#### 错误4：链接未建立
```
❌ 错误：文章创建后未建立双向链接
✅ 修正：重新执行Step 6，建立所有链接
```
