# 🎨 美文格式化工具 (prose-article-formatter)

## 📋 Skill 元数据
```yaml
name: prose-article-formatter
version: 1.0.0
description: 严格按照"美文赏析与教学通用模版"格式化散文文章,确保教学价值和互动性
author: AI Assistant
created: 2025-12-14
tags: [格式化, 模版, 排版, 教学设计]
```

## 🎯 核心功能

### 使用场景
当需要创建或格式化美文文章时,使用此skill确保符合统一的教学模版标准。

**触发条件:**
- 用户说"帮我格式化这篇文章"
- 用户说"按模版创建《作品名》"
- 用户说"规范化这篇美文"

## ⚠️ 铁律: 模版强制执行

### 为什么需要强制模版?
**血泪教训(来自真实案例):**
- 第五次执行时未遵循模版,缺少YAM

L、Emoji、教学板块
- 导致后期返工,全量重构

**原则:**
> 任何新收录文章必须100%符合模版,不得缺项!


**用户需求识别:**
- 用户说"格式化为小学美文" → 使用小学模版
- 用户说"格式化为初中美文" → 使用初中模版

- 未指定级别 → 询问用户或默认小学

**在开始格式化前,必须先读取对应级别的模版文件!**

## 🏗️ 执行流程

### Step 1: 识别级别并读取对应模版

**分级识别:**
- 根据用户需求确定目标级别(小学/初中/高中/大学/成人)
- 读取对应模版文件

**模版结构差异:**

#### 小学模版 📚
板块: 原文正文 → 美文赏析(写作特点+学习要点+优美词句) → 朗读指导 → 小练笔 → 知识小卡片

#### 初中模版 📖
板块: 原文正文 → 美文赏析(艺术手法+主题思想+精彩语段) → 朗读与批注 → 写作借鉴 → 文学常识 → 中考链接

#### 高中模版 📕
板块: 原文正文 → 文学赏析(艺术特色+主题意蕴+细读文本) → 批评视角 → 写作借鉴 → 作家作品 → 高考链接 → 知识网络

#### 大学模版 📘
板块: 原文正文 → 版本校勘 → 学术研究(文本细读+理论批评+风格学) → 语境研究 → 作家研究 → 互文性研究 → 研究综述 → 参考文献

#### 成人模版 📗
板块: 原文正文 → 深度品读(文学之美+人生启示+金句摘录) → 生活映照 → 作者世界 → 延伸阅读 → 品读札记 → 推荐人群 → 生活应用

### Step 2: 必需元素清单(7大板块)

#### 板块1: YAML 元数据 ✅
```yaml
---
title: {{文章标题}}
source: {{文章来源/出处}}
created: {{YYYY-MM-DD}}
tags:
  - 小学生美文
  - 散文
  - {{作者名}}
  - {{关键词1}}
  - {{关键词2}}
  - 完整版
---
```

**必需字段:**
- `title`: 作品标题
- `source`: 出处(如"散文集《踪迹》")
- `created`: 创建日期(YYYY-MM-DD格式)
- `tags`: 至少包含"小学生美文""完整版"

#### 板块2: 标题区 ✅
```markdown
# {{Emoji}} {{文章标题}}(完整版)
### —— {{作者名/风格/出处}}

---
```

**规则:**
- H1必须含Emoji(根据主题选择)
- 必须标注"(完整版)"
- H3格式: `—— 作者名`

**Emoji选择指南:**
- 写景: 🌅 🌿 🌸 🌊 ☁️ 🌙 ✨
- 动物: 🐱 🐶 🦢 🐦 🐝 🐜
- 植物: 🌳 🌺 🍀 🌾 🌻
- 抽象: 🎨 💡 🎭 📚 📝 ❤️
- 亲情: 👨‍👩‍👧 👵 🏠 💝
- 哲理: 💭 🌟 🔮 ⏰ 🌈

#### 板块3: 📖 原文正文 ✅
```markdown
## 📖 原文正文

{{开头Emoji组合}}

### {{Emoji}} {{小标题1}}
{{穿插Emoji的正文段落1...}}

### {{Emoji}} {{小标题2}}
{{穿插Emoji的正文段落2...}}

...
```

**规则:**
1. **必须为原文完整版**(非课文删减版)
2. **使用H3 + Emoji分段**(长文必须分段)
3. **正文必须穿插Emoji**(参考《春笋》标准)
4. **分段原则**:
   - 按内容逻辑分段(如:开头/发展/高潮/结尾)
   - 或按场景分段(如:早晨/中午/傍晚)
   - 或按对象分段(如:外形/习性/品格)

**正文穿插Emoji规范:**
```markdown
**开头Emoji组合**: 在正文开始前添加2-3个主题相关Emoji
**段落内穿插**: 在关键句子、情感表达、事物描述后添加Emoji
**密度统一**: 每2-3句话添加1个Emoji（所有级别统一标准）
**句子定义**: 以任何标点符号结尾的完整语义单元
**贴合内容**: Emoji必须与句子内容紧密相关
**无差异化**: 不采用克制规则，保持统一标准
```

**分段示例(《绿》):**
```markdown
### 🌿 惊诧于绿
我第二次到仙岩的时候,我惊诧于梅雨潭的绿了。

### 🛕 梅雨瀑与梅雨亭
梅雨亭正对着那条瀑布...

### 🌊 追捉神光
梅雨潭闪闪的绿色招引着我们...

### 💚 醉人的绿
——站在水边,望到那面...

### 🆚 对比与比拟(课文常删段落)
我曾见过北京什刹海拂地的绿杨...

### 🎀 女儿绿
我若能裁你以为带...
```

#### 板块4: 🌟 美文赏析 ✅
```markdown
## 🌟 美文赏析

### 🎨 写作特点
1. **{{特点1}}**: {{解释}}
2. **{{特点2}}**: {{解释}}

### 💡 学习要点
1. **{{要点1}}**: {{解释}}
2. **{{要点2}}**: {{解释}}

### ✨ 优美词句
- **{{词语1}}**: {{解释}}
- **{{词语2}}**: {{解释}}
```

**生成指南:**

**🎨 写作特点(分析修辞和结构):**
- 修辞手法: 比喻、拟人、排比、对比、夸张...
- 结构特点: 总分、对比、层递、首尾呼应...
- 表达方式: 记叙、描写、抒情、议论...

示例:
```markdown
1. **比喻与拟人**: 将绿比作"荷叶""裙幅""碧玉",化抽象为具体
2. **排比对比**: 通过什刹海、虎跑寺、西湖、秦淮河四个排比,烘托梅雨潭的绿
```

**💡 学习要点(适合学生模仿的技巧):**
- 观察方法
- 描写技巧
- 情感表达
- 主题升华

示例:
```markdown
1. **多角度描写**: 从视觉、触觉多角度描写"绿",让人仿佛能触摸到
2. **虚实相生**: 从实景写到虚幻的"舞女的带""盲妹的眼",意境深远
```

**✨ 优美词句(选择3-5个精彩词句):**
示例:
```markdown
- **"奇异的绿"**: 用"奇异"而非"美丽",强调绿的独特性
- **"醉人的绿"**: 反复出现,强调作者的沉醉和痴迷
- **"女儿绿"**: 创造性命名,赋予绿以生命和情感
```

#### 板块5: 🎭 朗读指导 ✅
```markdown
## 🎭 朗读指导

### 📢 语调建议
- {{建议1,如:轻柔舒缓}}
- {{建议2,如:高昂激越}}

### 🎪 节奏把握
- "{{关键句子}}" → {{节奏建议}}
```

**生成指南:**

**📢 语调建议(根据文章情感基调):**
- 轻柔舒缓: 写景抒情类
- 高昂激越: 励志进取类
- 深沉凝重: 哲理思考类
- 活泼明快: 童趣幽默类
- 悲伤忧郁: 怀念追忆类

示例:
```markdown
- 前半部分轻快跳跃,表现对绿的惊喜
- 中间部分舒缓绵长,表现对绿的沉醉
- 结尾部分深情款款,表现对绿的依恋
```

**🎪 节奏把握(选择2-3个关键句):**
示例:
```markdown
- "那醉人的绿呀!" → 语速稍慢,重读"醉人",表达沉醉之情
- "我若能裁你以为带,我将赠给那轻盈的舞女" → 节奏舒展,读出想象之美
```

#### 板块6: 🌈 小练笔 ✅
```markdown
## 🌈 小练笔

试着观察{{对象}},仿照本文写法,写一段话:
1. {{步骤1}}
2. {{步骤2}}
3. {{步骤3}}
```

**设计原则:**
1. **主题明确**: 仿写对象清晰
2. **步骤具体**: 3-5步,可操作
3. **难度适中**: 符合年龄段

**示例(《绿》):**
```markdown
试着观察一种你喜欢的颜色,仿照本文写法,写一段话:
1. 选择一种颜色(如红、黄、蓝)
2. 找到这种颜色的代表物(如红色的枫叶、夕阳、苹果)
3. 用比喻和拟人的方法,描写这种颜色的特点
4. 发挥想象,写出你对这种颜色的独特感受
5. 给这种颜色起一个美丽的名字
```

#### 板块7: 📚 知识小卡片 ✅
```markdown
## 📚 知识小卡片

**{{作者姓名}}**:
- {{身份/地位}}
- {{主要成就/作品}}
- {{本文背景/相关知识}}
```

**内容要点:**
1. 作者简介(50-100字)
2. 主要成就/代表作
3. 本文创作背景或相关知识

**示例(朱自清):**
```markdown
**朱自清**(1898-1948):
- 现代著名散文家、诗人、学者,清华大学教授
- 代表作:《背影》《荷塘月色》《春》《绿》等,被誉为"白话美文的典范"
- 《绿》写于1924年,收录于散文集《踪迹》,是作者游览浙江仙岩梅雨潭后的即兴之作,展现了朱自清散文"精致""清新"的艺术风格
```

## 🎨 Emoji使用规范

### 原则
1. **恰当性**: 符合文章主题和段落内容
2. **多样性**: 避免重复使用相同Emoji
3. **审美性**: 选择美观、寓意贴切的Emoji
4. **适度性**: 不过度使用,保持专业感

### 正文穿插Emoji详细规范

#### 参考标准:《春笋（完整版）》
**成功要素:**
- ✅ 开头有2-3个主题Emoji组合
- ✅ 正文中密度统一(每2-3句话1个)
- ✅ 句子定义：以任何标点符号结尾的完整语义单元
- ✅ Emoji与内容高度贴合
- ✅ 丰富视觉表现力
- ✅ 所有级别采用统一标准，无差异化
- ✅ 不影响阅读流畅性

#### 穿插位置指南

**1. 情感表达后添加Emoji:**
```markdown
"我的心中涌起了无限感慨 😔"
"我们都笑了 😄"
"母亲憔悴的脸上 😔"
```

**2. 事物描述后添加Emoji:**
```markdown
"金色的菜花 🌻、两行整齐的桑树 🌳"
"树上的嫩芽也密了 🌱、田里的冬水也咕咕地起着水泡 💧"
```

**3. 动作行为后添加Emoji:**
```markdown
"我蹲下来 🧎‍♂️、背起了母亲 👵"
"母亲悄悄地躲出去 🚶‍♀️"
```

**4. 环境氛围后添加Emoji:**
```markdown
"天气很好 😊"
"春天总算来了 🌸"
"在这满月的光里 🌕"
```

**5. 抽象概念后添加Emoji:**
```markdown
"感到了责任的重大 ⚖️"
"整个世界 🌍"
"生命的意义 💫"
```

**小学生美文 📚:**

- 风格: 🌸🌻🐢🦋🌈⭐等生动活泼

**高中生美文 📕:**

**大学生美文 📘:**
- 特点: 学术性、理性化
- 密度: 每4-5句话1个Emoji

**成人美文 📗:**
- 特点: 人生智慧、情感共鸣
- 密度: 每3-4句话1个Emoji

#### 穿插技巧和禁忌

**✅ 推荐做法:**
- 在标点符号后添加Emoji
- 选择与句子核心信息相关的Emoji
- 保持Emoji风格的一致性
- 优先使用系统原生支持的Emoji

**❌ 避免做法:**
- 不要在同一句话中添加过多Emoji
- 避免使用与内容无关的Emoji
- 不要在文言文部分添加现代Emoji
- 避免使用过于小众或显示异常的Emoji

#### 实践检查清单

**格式化完成后检查:**
```markdown
## 正文Emoji穿插检查清单
```

## 💡 常见问题处理

### Q1: 原文过长怎么办?
**A**: 分段要更细致
- 超过2000字: 至少5个分段
- 超过3000字: 至少7个分段
- 每个分段300-500字为宜

### Q2: 原文无明显分段怎么办?
**A**: 按内容逻辑强制分段
- 写景: 按空间顺序或时间顺序
- 状物: 按外形/习性/品格
- 记事: 按起因/经过/结果
- 抒情: 按情感层次递进

### Q3: 作者背景不详怎么办?
**A**: 简化知识小卡片
```markdown
**{{作者}}**:
- 近现代散文作家
- 本文选自{{出处}}
- {{简要说明文章价值}}
```

### Q4: 小练笔不知道写什么?
**A**: 通用模版
```markdown
试着仿照本文写法,写一段话:
1. 选择一个{{主题相关}}的对象
2. 仔细观察它的特点
3. 运用{{本文主要修辞手法}}进行描写
4. 表达你的真实感受
```
