## 🔍 完整预查重流程

### 第一步：读取并分析总索引

**执行内容：**
1. **读取总索引文件**：`d:\workspace\美文总索引.md`
2. **完整分析内容**：
   - 各文件夹现有文章统计
   - 按作家分类的作品分布
   - 按题材分类的文章覆盖
   - 跨文件夹重复关系记录
   - 总体收录进度情况

**分析重点：**
```markdown
- 小学生美文：现有篇数、主要作家、题材分布
- 初中生美文：现有篇数、主要作家、题材分布  
- 高中生美文：现有篇数、主要作家、题材分布
- 大学生美文：现有篇数、主要作家、题材分布
- 成人美文：现有篇数、主要作家、题材分布
- 重复关系：跨文件夹重复文章的记录
- 总体统计：文章总数、涉及作家数、完成度
```

### 第二步：识别重复风险

**执行内容：**
1. **作家重复风险**：
   - 已大量收录的作家标记
   - 高频出现作品的作家识别
   - 需要分散收录的作家清单

2. **作品重复风险**：
   - 同一作家多作品收录情况
   - 经典热门作品的收录密度
   - 跨文件夹重复的可能性

3. **题材重复风险**：
   - 题材覆盖不均衡分析
   - 过度集中的题材类型
   - 需要补充的题材缺口

**风险等级分类：**
```markdown
🔴 高风险：已达到重复上限（2篇）的作品
🟡 中风险：已收录1篇，需谨慎考虑重复收录
🟢 低风险：尚未收录，可优先考虑收录
```

### 第三步：规划非重复收集策略

**执行内容：**
1. **优先级排序**：
   - 优先收录低风险作品
   - 平衡不同作家的收录数量
   - 补充题材覆盖缺口

2. **避免重复策略**：
   - 标记高风险作品，避免重复收录
   - 为中风险作品制定替代方案
   - 建立作家收录平衡表

3. **收集清单制定**：
   - 确认目标级别
   - 制定推荐作家清单
   - 规划题材多样化收录

**策略输出示例：**
```markdown
## 🎯 {级别}美文收集策略

### ✅ 推荐收录（低风险）
- 未收录的作家作品
- 题材缺口补充作品
- 平衡收录需要的作品

### ⚠️ 谨慎考虑（中风险）  
- 已收录1篇的作家其他作品
- 需要建立跨文件夹链接的作品

### 🚫 避免收录（高风险）
- 已达到2篇上限的作品
- 明显重复的经典作品
```
