## 📊 删改类型统计

### 常见删改内容
1. **时代背景**(最常删)
   - 抗战、解放战争等历史背景
   - 家道中落、社会动荡等

2. **悲剧结局**(常删)
   - 人物死亡、离别等
   - 悲凉情感表达

3. **复杂情感**(常删)
   - 乡愁、孤独、忧伤等
   - 对社会的批判或反思

4. **哲理升华**(常删)
   - 深刻的人生思考
   - 对生命的感悟

### 删改原因分析
1. **篇幅限制**: 教材空间有限
2. **难度考虑**: 删去不适合学生的内容
3. **情感因素**: 删去过于悲伤或复杂的情感
4. **时代因素**: 删去特定历史背景

## 🔍 快速核查流程

### Step 1: 查清单
```
在本清单中搜索作品名 → 
  找到 → 查看删改情况 → 搜索完整版
  未找到 → 进行Step 2
```

### Step 2: 搜索版本差异
```
搜索: "{作品名} 课文与原文区别"
  有结果 → 根据结果找完整版
  无结果 → 进行Step 3
```

### Step 3: 对比字数
```
课文版字数 vs 网上找到的版本
  明显偏短 → 可能是删减版,继续搜索
  长度相近 → 可能是完整版,核查内容
```

### Step 4: 检查特征
```
检查是否包含:
  ✅ 完整开头(有背景介绍)
  ✅ 完整结尾(有情感升华)
  ✅ 复杂情感(不仅是浅层描写)
  ✅ 时代背景(如果是历史时期作品)
```
