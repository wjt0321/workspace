## 🎯 核心功能

### 使用场景
当需要为片段找到完整原文时,使用此skill确保收录的是**真正的完整版**而非课文节选版。

**触发条件:**
- 用户说"帮我找《作品名》的完整版"
- 用户说"这个片段的原文是什么"
- 用户说"为序号X找完整文章"

## ⚠️ 核心理念: 三步验证法

### 为什么需要强制验证?
**血泪教训(来自真实案例):**
1. 《燕子》→ 实为《海燕》节选,删去了乡愁部分
2. 《白鹅》→ 删去了抗战背景和被吃掉的悲剧结尾
3. 《背影》→ 部分版本删减了家庭背景描写

**原则:**
> 宁可不收录,也不能收录删减版!

## 🏗️ 执行流程

### Step 1: 识别目标级别并准备信息

**分级系统(5个级别):**
```yaml
小学生美文: d:\workspace\小学生美文\
初中生美文: d:\workspace\初中生美文\
高中生美文: d:\workspace\高中生美文\
大学生美文: d:\workspace\大学生美文\
成人美文: d:\workspace\成人美文\
```

**用户需求识别:**
- 自动识别用户指定的目标文件夹
- 根据级别调整搜索策略(详见下方)

**从片段中提取关键信息:**
```yaml
作品名: "绿"
作者名: "朱自清"
关键句: "梅雨潭闪闪的绿色招引着我们"
出处提示: "散文集《踪迹》"(如果片段中有提及)
目标级别: "小学生美文"(或其他)
```

### Step 2: 三步搜索策略

#### 第一步: 搜索完整版关键词
**搜索组合(按优先级):**
1. `"{作品名}" "{作者}" 原文完整版`
2. `"{作品名}" "{作者}" 未删减版`
3. `"{作品名}" "{作者}" 文集版`
4. `"{作品名}" "{作者}" 全文`

**示例:**
```
"绿" "朱自清" 原文完整版
"白鹅" "丰子恺" 未删减版
```

#### 第二步: 版本对比(🚨最关键)
**强制核查清单:**

如果作品是**著名课文**,**必须**执行以下搜索:
1. `"{作品名}" 课文与原文区别`
2. `"{作品名}" 教材删改`
3. `"{作品名}" 课文版 vs 原文版`
4. `"{作品名}" 删减内容`

**判断标准:**
- ✅ 如果搜索结果显示"无删改" → 继续
- ⚠️ 如果搜索结果显示"有删减" → **必须找到完整版**
- ⛔ 如果搜索结果显示"大量删改" → **警告用户,优先找完整版**

#### 第三步: 版本验证
**验证维度:**

1. **字数验证**
   - 完整版通常比课文版更长
   - 对比字数,如果偏短需警惕

2. **结构验证**
   - 检查是否有开头/结尾
   - 检查是否有背景说明
   - 检查是否有情感完整性

3. **特征验证**
   - 查看是否包含"历史背景"段落
   - 查看是否有"悲剧/深刻"结局
   - 查看是否有"复杂情感"表达

## 🎯 版本优先级(按级别调整)

### 基础优先级排序
1. **最优 ⭐⭐⭐**: 作家文集/作品集中的版本
   - 如: 《朱自清散文集》《老舍文集》
   - 特征: 通常注明"收录时未改动"

2. **次优 ⭐⭐**: 文学网站标注"完整版"的版本
   - 如: "朱自清《绿》原文完整版"
   - 特征: 明确标注"未删减""完整版"

3. **可用 ⭐**: 权威出版社图书扫描版
   - 如: 人民文学出版社、作家出版社
   - 特征: 有明确出版信息

4. **避免 ⛔**: 教材、课文、节选版本
   - 如: "人教版""语文课本"
   - 特征: 标注"节选""课文"

### 分级调整

**小学/初中/高中**:
- 强制执行三步验证法
- 必须排除课文删减版
- 必须搜索"课文与原文区别"

**大学**:
- 要求版本校勘
- 必须注明版本来源(如"人民文学出版社2018年版")
- 如有多个版本,要说明差异

**成人**:
- 灵活处理版本问题
- 注重可读性和完整性
- 如无明显删改,可适当放宽

## 🎓 特殊情况处理

### 情况1: 多个版本差异不大
- 选择: 最权威的出版社版本
- 标注: 注明版本来源

### 情况2: 原文过长(>5000字)
- 评估: 是否适合完整收录
- 选择: 收录完整版 或 选择适当长度的节选(但必须标注"节选")

### 情况3: 原文使用繁体字
- 处理: 转换为简体中文
- 标注: 注明"已转简体"

### 情况4: 原文有多个版本(作者多次修订)
- 选择: 作者最后定稿版本
- 标注: 注明版本年份

## 💡 使用技巧

### 快速判断是否为删减版
1. **看标题**: 有"节选""选自"字样 → 警惕
2. **看来源**: 标注"教材""课文" → 必查
3. **看结构**: 突然开头/结尾 → 可疑
4. **看长度**: 明显过短 → 核查

创建 `version_diff_checklist.md` 文件,记录:
