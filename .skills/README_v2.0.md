# 📚 美文收集与整理Skills系统 v2.0

> **重大更新**: 从单一"小学生美文"扩展为**5级完整体系**，覆盖小学→初中→高中→大学→成人全阶段！

---

## 🎯 系统概述

### v2.0 核心升级

| 维度 | v1.0 | v2.0 |
|------|------|------|
| **级别** | 单一(小学) | 5级(小学/初中/高中/大学/成人) |
| **模版** | 1个通用模版 | 5个专属模版 |
| **Skills支持** | 单级别 | 全级别+智能识别 |
| **工作流** | 线性 | 多分支+跨级 |
| **适用场景** | 小学教育 | 全年龄段+多场景 |

---

## 📂 分级体系总览

### 5个级别 × 5个特性

| 级别 | 📚 小学 | 📖 初中 | 📕 高中 | 📘 大学 | 📗 成人 |

## 🔄 标准工作流程（重要！）

**⚠️ 每次收集美文时，必须严格按照以下6个步骤依次执行，不可跳过或颠倒顺序！**

### 6步标准流程

```
Step 1 → Step 2 → Step 3 → Step 4 → Step 5 → Step 6
```

| 步骤 | Skill名称 | 功能 | 关键输出 |
|------|----------|------|----------|
| 1️⃣ | prose-snippet-collector | 收集片段 | `{级别}美文精选100篇.md` |
| 2️⃣ | prose-index-manager | 更新索引 | `美文收集索引.md` |
| 3️⃣ | prose-fulltext-hunter | 搜索原文（三步验证） | 完整版验证 |
| 4️⃣ | prose-article-formatter | 格式化文章 | `{序号}.{作品}-{作者}.md` |
| 5️⃣ | prose-deduplicator | 去重检查 | 去重验证报告 |
| 6️⃣ | prose-link-weaver | 建立链接 | 双向链接完成 |

### 一键命令
```
"参考.skill，帮我收集N篇{级别}美文"
```

**详细文档**：[[标准工作流程.md]]

## 🔧 6个Skills全面升级

### 1. 📚 prose-snippet-collector (片段收集器)
**v2.0 新特性**:
- ✅ 自动识别目标级别(小学/初中/高中/大学/成人)
- ✅ 根据级别调整片段长度(150-500字)
- ✅ 智能推荐合适的作家和风格
- ✅ 分级存储到对应文件夹

**使用示例**:
```bash
"帮我收集10个初中美文片段"           # 200-400字，注重哲理
"为高中美文收集6个京派作家片段"       # 300-500字，注重深度
"为成人美文收集治愈系散文"           # 不限长度，注重共鸣
```

---

### 2. 🔍 prose-fulltext-hunter (原文搜索器)
**v2.0 新特性**:
- ✅ 根据级别调整搜索策略
- ✅ 小学/初中/高中强制三步验证(课文删改检查)
- ✅ 大学版增加版本校勘功能
- ✅ 成人版灵活处理版本问题

**使用示例**:
```bash
"找《白鹅》完整版，用于小学美文"     # 强制查删改
"找《受戒》完整版，用于大学美文"     # 需版本校勘
"找《清欢》完整版，用于成人美文"     # 注重可读性
```

---

### 3. 🎨 prose-article-formatter (格式化工具)
**v2.0 新特性**:
- ✅ 自动读取对应级别的模版
- ✅ 5套模版自适应切换
- ✅ 根据级别生成差异化板块内容
- ✅ 智能Tags分类

**级别模版差异**:
```yaml
小学模版: 原文 → 赏析 → 朗读 → 练笔 → 卡片
初中模版: 原文 → 赏析 → 批注 → 借鉴 → 常识 → 中考
高中模版: 原文 → 赏析 → 批评 → 借鉴 → 作品 → 高考 → 网络
大学模版: 原文 → 校勘 → 研究 → 语境 → 作家 → 互文 → 综述 → 文献
成人模版: 原文 → 品读 → 映照 → 世界 → 阅读 → 札记 → 人群 → 应用
```

**使用示例**:
```bash
"将《散步》格式化为初中美文"
"将《我与地坛》格式化为高中美文"
"将《目送》格式化为成人美文"
```

---

### 4. 📇 prose-index-manager (索引管理器)
**v2.0 新特性**:
- ✅ 分级管理5个索引文件
- ✅ 自动识别文章所属级别
- ✅ 跨级别序号独立管理
- ✅ 统一的序号分配策略

**使用示例**:
```bash
"更新初中美文索引"
"检查高中美文序号冲突"
"查看成人美文收录统计"
```

---

### 5. 🔗 prose-link-weaver (链接编织器)
**v2.0 新特性**:
- ✅ 自动识别文章所属级别
- ✅ 在对应级别内建立双向链接
- ✅ 支持跨级别引用(可选)
- ✅ 批量链接维护

**使用示例**:
```bash
"为初中美文建立双向链接"
"检查高中美文断裂链接"
"批量更新所有级别的链接"
```

---

### 6. 🔍 prose-deduplicator (去重检查器)
**v2.0 新特性**:
- ✅ 同级去重(在单一级别内检查)
- ✅ 跨级去重(检查是否在多个级别重复)
- ✅ 智能建议(推荐更合适的收录级别)
- ✅ 全局视图(5级整体清洁度)

**使用示例**:
```bash
"检查初中美文重复"              # 单级检查
"全局去重检查"                  # 跨5级检查
"建议《背影》的最佳收录级别"     # 智能推荐
```

## 🎯 典型工作流

### 场景1: 新用户从初中开始收录

```bash
步骤1: "帮我收集10个初中美文片段"
      → prose-snippet-collector 识别级别(初中)
      → 选择合适作家(鲁迅、史铁生、汪曾祺)
      → 提取200-400字片段
      → 存储到 d:\workspace\初中生美文\

步骤2: "为序号5《散步》找完整版"
      → prose-fulltext-hunter 搜索+验证
      → 确认无删减
      → 输出完整原文

步骤3: "将《散步》格式化为初中美文"
      → prose-article-formatter 读取初中模版
      → 生成7大板块(含批注+中考链接)
      → 创建 18.散步-莫怀戚.md

步骤4: 自动执行
      → prose-index-manager 更新索引
      → prose-link-weaver 建立双向链接
      → prose-deduplicator 去重检查

完成: 初中美文首篇收录成功✅
```

---

### 场景2: 同一文章多级别收录

**目标**: 将朱自清《背影》收录到小学、初中、高中三个级别

```bash
# 小学版 - 侧重父爱和细节
"将《背影》格式化为小学美文，侧重父爱描写和朗读指导"
→ 输出: 15.背影-朱自清.md (小学版)
→ 板块: 原文+赏析(细节描写)+朗读+练笔(写父母)+卡片

# 初中版 - 增加批注和情感分析
"将《背影》格式化为初中美文，增加批注和中考要点"
→ 输出: 25.背影-朱自清.md (初中版)
→ 板块: 原文+赏析(情感变化)+批注+借鉴+中考(记叙文手法)

# 高中版 - 文学批评和细读
"将《背影》格式化为高中美文，注重文学批评和细读文本"
→ 输出: 48.背影-朱自清.md (高中版)
→ 板块: 原文+赏析(细读关键段)+批评(多元解读)+高考+网络

结果: 同一文章在3个级别有不同深度的收录✅
```

---

### 场景3: 全局质量检查

```bash
用户: "全局去重检查，检查所有5个级别"

系统执行:
1. prose-deduplicator 扫描5个文件夹
2. 同级检查(各级别内部)
3. 跨级检查(5级之间)
4. 生成报告:
   ✅ 小学: 48篇，无重复
   ⚠️ 初中: 10篇，发现1个与小学重复
   ✅ 高中: 8篇，无重复
   ✅ 大学: 5篇，无重复
   ✅ 成人: 12篇，无重复
5. 智能建议:
   - 《春》在小学和初中都有，建议保留两者(难度不同)
   - 建议为初中版增加批注板块以区分

用时: 约8-10分钟
输出: 详细去重报告 + 处理建议
```

## ⚠️ 核心规则(三大铁律)

### 铁律1: 作家筛选
✅ **必须**: 近现代中文散文大家  
❌ **禁止**: 古典作者、外文作者、网络作家

### 铁律2: 完整版验证
✅ **必须**: 原文完整版(尤其小学/初中/高中)  
❌ **禁止**: 课文删减版  
🔍 **强制**: 搜索"课文与原文区别"

### 铁律3: 模版执行
✅ **必须**: 100%符合对应级别模版  
❌ **禁止**: 缺板块、缺Emoji、缺YAML  
📝 **检查**: 完整性清单逐项核对

### v2.0 新增优势

1. **全年龄覆盖** - 从小学到成人，满足全生命周期阅读需求
