## 🎯 核心理念

**Agent Harness = Tools + Knowledge + Observation + Action + Permissions**

- Model 提供智能
- Harness 提供"手、眼、记忆和安全边界"

## 📂 源码结构

```
openharness/
├── engine/          # Agent Loop 核心
│   ├── query_engine.py   # 主循环入口
│   ├── query.py          # QueryContext + run_query
│   ├── cost_tracker.py   # Token/成本追踪
│   └── stream_events.py  # 流式事件
├── tools/           # 43+ 工具
│   ├── base.py           # BaseTool + ToolRegistry
│   ├── bash_tool.py
│   ├── file_read_tool.py / file_write_tool.py / file_edit_tool.py
│   ├── web_fetch_tool.py / web_search_tool.py
│   ├── mcp_tool.py
│   └── ...
├── skills/          # Skill 加载
│   ├── loader.py        # 加载逻辑
│   ├── registry.py
│   └── types.py
├── memory/          # 持久化记忆
│   ├── manager.py       # 记忆文件管理
│   └── ...
├── permissions/     # 权限系统
│   ├── checker.py       # PermissionChecker
│   └── modes.py        # PermissionMode 枚举
├── coordinator/    # 多 Agent 协调
│   ├── coordinator_mode.py
│   └── agent_definitions.py
├── plugins/        # 插件系统
├── api/            # API 客户端（Anthropic/OpenAI/Copilot）
├── channels/       # 渠道集成（飞书/Discord/Slack 等）
└── ui/             # React TUI + 终端 UI

src/openharness/tools/base.py
```

## 🔑 关键文件速查

| 文件 | 作用 |
|------|------|
| `engine/query_engine.py` | Agent 主循环，submit_message/continue_pending |
| `engine/query.py` | run_query 核心，AsyncIterator[StreamEvent] |
| `tools/base.py` | BaseTool 抽象类 + ToolRegistry |
| `skills/loader.py` | Skill 加载，支持 .md 文件 |
| `memory/manager.py` | 记忆文件管理，add_memory_entry |
| `permissions/checker.py` | 权限决策，PathRule glob 匹配 |
| `coordinator/coordinator_mode.py` | TeamRegistry + 多 Agent 协作 |
