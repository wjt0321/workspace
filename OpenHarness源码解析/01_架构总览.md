# 01 - 架构总览

## 整体架构图

```
User Prompt
    ↓
CLI (oh) / React TUI
    ↓
QueryEngine (主循环入口)
    ↓
QueryContext (上下文封装)
    ↓
run_query (核心异步循环)
    ↓
┌─────────────────────────────────┐
│  API Client (流式)              │
│  - Anthropic (默认)             │
│  - OpenAI 兼容                 │
│  - GitHub Copilot              │
└─────────────────────────────────┘
    ↓ (tool_use)
ToolRegistry → PermissionChecker → HookExecutor
    ↓
Tool Execution (43+ 工具)
    ↓
StreamEvent 产出
```

## 核心组件

### 1. QueryEngine（引擎核心）

**位置：** `src/openharness/engine/query_engine.py`

```python
class QueryEngine:
    def __init__(self, *,
        api_client,           # API 客户端
        tool_registry,         # 工具注册表
        permission_checker,    # 权限检查器
        cwd, model, system_prompt,
        max_tokens, max_turns,
        permission_prompt,
        hook_executor,
    ):
```

**关键方法：**
- `submit_message(prompt)` → `AsyncIterator[StreamEvent]` - 提交用户消息并执行循环
- `continue_pending()` - 继续中断的工具循环
- `has_pending_continuation()` - 检查是否有待续的工具结果

**消息管理：**
- `_messages: list[ConversationMessage]` - 内存中的对话历史
- `load_messages()` - 替换对话历史
- `clear()` - 清空对话历史

### 2. ToolRegistry（工具注册表）

**位置：** `src/openharness/tools/base.py`

```python
class BaseTool(ABC):
    name: str
    description: str
    input_model: type[BaseModel]  # Pydantic 模型

    async def execute(self, arguments: BaseModel, context: ToolExecutionContext) -> ToolResult:
        """执行工具"""

    def is_read_only(self, arguments: BaseModel) -> bool:
        """是否只读工具"""

    def to_api_schema(self) -> dict:
        """转换为 Anthropic API 格式"""

class ToolRegistry:
    def register(self, tool: BaseTool): ...
    def get(self, name: str) -> BaseTool | None: ...
    def list_tools() -> list[BaseTool]: ...
    def to_api_schema() -> list[dict]: ...
```

**工具分类（43+）：**

| 类别 | 工具 |
|------|------|
| 文件操作 | Read, Write, Edit, Glob, Grep |
| Shell | Bash |
| 搜索 | WebFetch, WebSearch, ToolSearch, LSP |
| Notebook | NotebookEdit |
| Agent | Agent, SendMessage, TeamCreate/Delete |
| Task | TaskCreate/Get/List/Update/Stop/Output |
| MCP | MCPTool, ListMcpResources, ReadMcpResource |
| Mode | EnterPlanMode, ExitPlanMode, Worktree |
| Schedule | CronCreate/List/Delete, RemoteTrigger |
| Meta | Skill, Config, Brief, Sleep, AskUser |
