## 4.2 启动流程

```typescript
// 1. 性能埋点
profileCheckpoint('main_tsx_entry')

// 2. 前置导入（并行）
startMdmRawRead()           // MDM 设置读取
startKeychainPrefetch()     // 密钥链预读取

// 3. 初始化
await init()

// 4. 解析命令行
const program = new Command()
program
  .name('claude')
  .version('2.1.88')
  .option('-p, --print', 'Print mode')
  .option('-o, --output', 'Output file')
  // ...

// 5. 加载配置
await loadRemoteManagedSettings()
await refreshPolicyLimits()

// 6. 启动 REPL
launchRepl()
```

## 4.3 关键初始化步骤

### 1. MDM 读取（ macOS 移动设备管理）

```typescript
// 并行读取 MDM 配置
startMdmRawRead()

// 用于企业环境下的配置管理
interface MdmSettings {
  apiKey?: string
  config?: string
  policies?: PolicyLimits
}
```

### 2. 密钥链预读取

```typescript
// 预读取 API 密钥，避免每次请求时读取
startKeychainPrefetch()
```

### 3. 远程托管设置

```typescript
// 企业环境可能通过远程配置管理
await loadRemoteManagedSettings()

// 托管设置可以：
// - 锁定 API Key
// - 限制可用模型
// - 强制开启审计日志
```
