## 项目概述

此工作空间是一个全面的**AI 和自动化教程资源中心**，专注于智能系统、工作流自动化和 AI 应用开发。它被组织成一个学习中心，涵盖多个低代码 AI 平台和开发框架。

## 目录结构

**主要学习资源：**
- `智能体学习/` - Hello-Agents 教程系统（全面的智能体开发指南）
- `coze教程资源库/` - Coze 平台教程和资源
- `dify教程资源库/` - Dify 平台教程和资源
- `n8n教程资源库/` - n8n 工作流自动化教程
- `自动化工作流参考资料库/` - 全面的工作流自动化参考资料
- `n8n实例/` - 实用的 n8n 工作流示例
- `Python资源库/` - Python 编程资源

**关键领域：**
- 智能体理论和演进历史
- 经典范式（ReAct、Plan-and-Solve、Reflection）
- 低代码平台使用（Coze、Dify、n8n）
- 自定义智能体框架开发
- 高级功能：记忆系统、上下文工程、通信协议

## 技术栈

**AI/智能体开发：**
- Hello-Agents 框架、LLM 集成、多智能体系统
- MCP（模型上下文协议）、A2A、ANP 通信协议
- 使用 Socket.io 的实时应用

**工作流平台：**
- n8n、Coze、Dify（低代码自动化平台）

**前端：** React 18.2.0、Material-UI、用于数据可视化的 Recharts
**后端：** Node.js/Express、MongoDB、PostgreSQL、Redis
**开发工具：** Babel、ESLint、Jest、Nodemon

## 开发命令

**后端开发（分析与商业智能平台）：**
```bash
npm start           # 生产服务器
npm run dev         # 使用 nodemon 的开发服务器
npm test            # 运行 Jest 测试
npm run lint        # ESLint 代码检查
npm run build       # Babel 转译
```

**前端开发（React 仪表板）：**
```bash
npm start           # React 开发服务器
npm run build       # 生产构建
npm test            # React 测试
npm run lint        # ESLint 检查
npm run lint:fix    # 自动修复代码风格问题
```
