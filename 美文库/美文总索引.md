1. **按作者检索**：在Obsidian中搜索作者姓名，如"朱自清"、"鲁迅"
2. **按主题检索**：使用标签筛选，如"#散文"、"#亲情类"
3. **按级别检索**：直接访问对应文件夹
4. **关键词搜索**：在片段文件中搜索关键词定位相关文章

### 📝 学习建议

#### 片段学习法
1. 先阅读片段文件中的精彩段落
2. 点击链接跳转到完整文章
3. 结合赏析板块深入理解
4. 运用写作借鉴板块进行仿写

#### 主题阅读法
1. 确定学习主题（如"思乡"、"母爱"、"自然"）
2. 在总索引中查找相关主题文章
3. 对比不同作者的写作手法
4. 总结归纳形成自己的写作素材库
