# 小学生美文项目操作规范

## 🎯 项目概述
本项目旨在构建一个高质量、结构化的小学生美文阅读资源库。通过精选经典美文，提供科学的分类、详细的导读和便捷的索引，辅助小学生提升阅读审美与写作能力。

---

## 📂 文件结构体系

### 1. 核心文件
| 文件类型 | 命名规则 | 说明 | 示例 |
| :--- | :--- | :--- | :--- |
| **内容主文件** | `[名称]（[版本]）.md` | 存放文章正文、推荐理由及学习指导 | `小学生美文精选100篇（完整版）.md` |
| **扩展内容文件** | `[名称].md` | 存放新增或补充的文章集合 | `美文收集.md` |
| **索引文件** | `[对应内容文件名]索引.md` | 提供行号定位、分类检索功能 | `小学生美文精选100篇索引.md` |
| **规范文件** | `项目操作规范.md` | 项目执行的统一标准（本文档） | - |

### 2. 目录组织
所有文件均位于项目根目录 `d:\workspace\小学生美文\` 下，保持扁平化结构，便于管理。

## 📝 内容收录标准

### 1. 选文原则
- **经典性**：优先选择名家名篇及教材选文（如老舍、巴金、冰心等作品）。
- **适读性**：语言优美，生字词适量，符合小学各学段认知水平。
- **多样性**：覆盖写景、状物、叙事、哲理等多种题材。
- **正能量**：主题积极向上，蕴含教育意义或情感熏陶价值。

### 2. 排版格式规范
每篇文章必须严格遵守以下 Markdown 格式：

```markdown
#### **序号. 《标题》 - 作者**

> 正文内容...（段落之间空一行）
>
> 正文内容...

**【推荐理由】** 简要说明文章特色、写作手法或教育价值。（100字以内）

---
```

**注意要点**：
- 标题使用 H4 (`####`) 加粗。
- 正文使用引用块 (`> `)。
- 分隔线 (`---`) 用于区分不同文章。

### 3. 分类体系
文章应归入以下四大类之一：
1.  **写景状物经典**：描绘自然风光、城市景观。
2.  **动植物描写精品**：描写动物、植物的形态习性。
3.  **亲情记事类文章**：记述家庭亲情、童年趣事、校园生活。
4.  **哲理感悟类文章**：蕴含人生哲理、成长感悟、励志故事。

## 🗂️ 索引维护规则

### 1. 索引文件结构
索引文件必须包含以下部分：
- **文件信息**：源文件路径、创建日期、文章总数。
- **内容分类索引**：按四大分类列出文章标题及**行号**。
- **按年级/主题推荐**：提供不同维度的快速定位列表。
- **使用说明**：指导用户如何使用索引。

### 2. 行号定位
- 必须准确标注每篇文章在内容文件中的**起始行号**（即标题所在行）。
- 格式示例：`1. 《草原》 - 老舍【20】`

### 3. 更新机制
- 每次修改内容文件（新增/删除/移动文章）后，**必须**同步更新对应的索引文件。
- 确保索引中的行号与实际文件保持一致。

## 🔄 标准工作流程

### 1. 新增文章流程
1.  **需求分析**：确定需要补充的题材或主题。
2.  **查重检索**：
    - 检查 `小学生美文精选100篇索引.md`。
    - 检查 `美文收集索引.md`。
    - **严禁收录重复文章**。
3.  **内容获取**：从权威渠道获取原文，核对作者和文本准确性。
4.  **格式化编辑**：按照“内容收录标准”进行排版。
5.  **写入文件**：将新文章追加到相应的内容文件中（如 `美文收集.md`）。
6.  **更新索引**：计算新文章行号，更新对应的索引文件。

### 2. 质量检查清单
- [ ] 标题格式是否统一？
- [ ] 正文是否有错别字或排版错误？
- [ ] 推荐理由是否精炼准确？
- [ ] 是否与现有库中文章重复？
- [ ] 索引链接（行号）是否跳转准确？

### 3. 扩写文章流程
当需要将现有内容文件进行扩写时，应遵循以下流程：
1.  **需求分析**：确定扩写的目标数量和题材分布。
2.  **查重检索**：
    - 检查 `小学生美文精选100篇索引.md`。
    - 检查 `美文收集索引.md`。
    - **严禁收录重复文章**。
3.  **内容获取**：从权威渠道获取原文，核对作者和文本准确性。
4.  **格式化编辑**：按照“内容收录标准”进行排版。
5.  **写入文件**：将新文章插入到相应的分类中，确保分类结构均衡。
6.  **序号修正**：确保所有文章序号从1开始连续排列。
7.  **学习指导更新**：根据当前篇目更新学习指导部分，确保推荐的文章是当前文件中存在的。
8.  **更新索引**：计算所有文章的行号，更新对应的索引文件，确保索引准确反映当前文章情况。

### 4. 重复篇目替换流程
当发现内容文件中存在与其他文件重复的篇目时，应遵循以下流程：
1.  **重复检测**：对比现有文件与目标文件，找出重复篇目。
2.  **篇目筛选**：选择内容优质、适合小学生阅读的新篇目替换重复篇目。
3.  **格式调整**：按照“内容收录标准”对新篇目进行排版。
4.  **替换编辑**：将重复篇目替换为新篇目，确保序号连续。
5.  **学习指导更新**：根据替换后的篇目更新学习指导部分。
6.  **更新索引**：更新索引文件，确保索引准确反映当前文章情况。

## 📚 学习指导编写规范
在内容文件的末尾，应包含“学习指导”部分，结构如下：
1.  **按年级推荐**：低、中、高年级的重点推荐篇目及学习重点。
2.  **学习活动建议**：基础、进阶、深度的阅读活动设计。
3.  **教学方法建议**：针对家长或老师的指导策略。
4.  **使用建议**：通用的阅读建议。
