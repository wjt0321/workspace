## 🚀 快速开始

### 1️⃣ 克隆仓库

```bash
git clone https://github.com/yourusername/vibe-coding-tutorial.git
cd vibe-coding-tutorial
```

### 2️⃣ 安装 Obsidian（推荐）

本教程使用 Obsidian 格式编写，推荐安装 Obsidian 以获得最佳阅读体验：

1. 下载 [Obsidian](https://obsidian.md)
2. 打开仓库文件夹作为 Vault
3. 安装推荐插件：
   - **Dataview**：查询和展示
   - **Templater**：模板功能
   - **Kanban**：看板视图

### 3️⃣ 开始学习

按照章节顺序学习，建议：
- 📖 先阅读理论章节（1-4章）
- 💻 再动手实战（5-8章）
- 🎯 最后掌握进阶技巧（9-10章）
