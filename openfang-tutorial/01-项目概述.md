# OpenFang 项目概述

## 什么是 OpenFang？

OpenFang 是一个**开源智能体操作系统（Agent Operating System）**，使用 Rust 语言从零构建。它不是聊天机器人框架，不是 Python 包装的 LLM，也不是"多智能体编排器"——它是一个完整的自主智能体操作系统。

> [!important] 核心理念
> 传统智能体框架等待你输入内容。OpenFang 运行**自主智能体为你工作**——按计划、全天候、构建知识图谱、监控目标、生成线索、管理社交媒体、并向仪表板报告结果。

## 核心创新：Hands（手）

Hands 是 OpenFang 的核心创新——预构建的自主能力包，独立运行、按计划执行、无需你手动提示。

> [!quote] 设计理念
> "传统智能体等待你输入。Hands 为你工作。"

每个 Hand 包含：

| 组件 | 说明 |
|------|------|
| `HAND.toml` | 清单文件，声明工具、设置、要求和仪表板指标 |
| System Prompt | 多阶段操作手册（500+字专家流程） |
| `SKILL.md` | 运行时注入上下文的领域专业知识参考 |
| Guardrails | 敏感操作的审批门控 |

## 内置 Hands 一览

| Hand | 功能描述 |
|------|----------|
| **Clip** | YouTube视频处理：下载、识别精彩片段、制作竖屏短视频、添加字幕和缩略图、AI配音、发布到Telegram/WhatsApp。8阶段流水线，支持FFmpeg + yt-dlp + 5种STT后端 |
| **Lead** | 每日运行：发现符合ICP的潜在客户、网络研究丰富信息、0-100评分、去重、CSV/JSON/Markdown格式输出。随时间构建ICP画像 |
| **Collector** | OSINT级情报收集：监控目标（公司/人物/主题）、变化检测、情感追踪、知识图谱构建、重要变化时发出警报 |
| **Predictor** | 超级预测引擎：收集多源信号、构建校准推理链、带置信区间预测、使用Brier分数追踪准确性。支持逆向模式 |
| **Researcher** | 深度自主研究：交叉引用多源、CRAAP标准评估可信度、APA格式引用报告、多语言支持 |
| **Twitter** | 自主Twitter/X账号管理：7种轮换内容格式、最佳时间发布、回复提及、追踪性能指标。有审批队列机制 |
| **Browser** | 网页自动化：导航网站、填写表单、点击按钮、处理多步骤工作流。Playwright桥接、会话持久化。**强制购买审批** |

## 适用场景

> [!tip] 推荐使用场景
> - 🔍 **市场研究**：Researcher + Collector 自动收集分析行业动态
> - 📊 **销售线索**：Lead 自动发现和评分潜在客户
> - 📱 **内容创作**：Clip + Twitter 自动化内容生产和发布
> - 🎯 **竞争监控**：Collector 持续监控竞争对手动态
> - 📈 **趋势预测**：Predictor 分析信号并做出预测
