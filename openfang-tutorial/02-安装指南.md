# OpenFang 安装指南

## 系统要求

### 支持平台

| 平台 | 架构 | 状态 |
|------|------|------|
| macOS | x86_64, ARM64 (M1/M2) | ✅ 支持 |
| Linux | x86_64, ARM64 | ✅ 支持 |
| Windows | x86_64 | ✅ 支持 |

### 最低要求

```
┌─────────────────────────────────────┐
│         系统最低要求                │
├─────────────────────────────────────┤
│ CPU        │ 2核心及以上            │
│ 内存       │ 4GB RAM（推荐8GB+）    │
│ 磁盘       │ 100MB可用空间          │
│ 网络       │ 需要互联网连接         │
└─────────────────────────────────────┘
```

## 安装方法

### macOS / Linux

使用安装脚本一键安装：

```bash
curl -fsSL https://openfang.sh/install | sh
```

### Windows

使用 PowerShell 安装：

```powershell
irm https://openfang.sh/install.ps1 | iex
```

### 手动安装

从 GitHub Releases 下载对应平台的二进制文件：

```
https://github.com/RightNow-AI/openfang/releases
```

选择对应版本：
- `openfang-x86_64-apple-darwin.tar.gz` (macOS Intel)
- `openfang-aarch64-apple-darwin.tar.gz` (macOS Apple Silicon)
- `openfang-x86_64-unknown-linux-gnu.tar.gz` (Linux x64)
- `openfang-x86_64-pc-windows-msvc.zip` (Windows)

## 安装后配置

### 1. 初始化

安装完成后，运行初始化命令：

```bash
openfang init
```

初始化过程会：
- 创建配置目录（`~/.openfang/`）
- 引导配置 LLM 提供商
- 生成默认配置文件

> [!tip] 配置目录
> - **macOS/Linux**: `~/.openfang/`
> - **Windows**: `%USERPROFILE%\.openfang\`

### 3. 配置文件示例

```toml
# ~/.openfang/config.toml

[provider]
default = "anthropic"

[provider.anthropic]
api_key = "${ANTHROPIC_API_KEY}"
model = "claude-sonnet-4-20250514"

[provider.openai]
api_key = "${OPENAI_API_KEY}"
model = "gpt-4o"

[provider.ollama]
base_url = "http://localhost:11434"
model = "llama3"

[server]
host = "127.0.0.1"
port = 4200

[dashboard]
enabled = true
```

### 4. 环境变量

推荐使用环境变量管理 API 密钥：

```bash
# ~/.bashrc 或 ~/.zshrc
export ANTHROPIC_API_KEY="your-key-here"
export OPENAI_API_KEY="your-key-here"
export GEMINI_API_KEY="your-key-here"
```

Windows PowerShell：

```powershell
# 设置环境变量
[Environment]::SetEnvironmentVariable("ANTHROPIC_API_KEY", "your-key-here", "User")
```

## 启动服务

### 启动守护进程

```bash
openfang start
```

服务启动后：
- **Dashboard**: http://localhost:4200
- **API**: http://localhost:4200/v1

### 后台运行

```bash
# 以守护进程方式运行
openfang start --daemon

# 查看状态
openfang status

# 停止服务
openfang stop
```

## 验证安装

### 检查版本

```bash
openfang --version
# 输出: openfang 0.1.0
```

### 健康检查

```bash
curl http://localhost:4200/health
```

### 测试API

```bash
curl -X POST http://localhost:4200/v1/chat/completions \
  -H "Content-Type: application/json" \
  -d '{
    "model": "default",
    "messages": [{"role": "user", "content": "Hello!"}]
  }'
```

## 故障排除

### 常见问题

> [!warning] 端口占用
> 如果4200端口被占用：
> ```bash
> openfang start --port 4201
> ```

> [!warning] 权限问题
> Linux/macOS 可能需要：
> ```bash
> chmod +x $(which openfang)
> ```

> [!warning] API Key 无效
> 检查环境变量是否正确设置：
> ```bash
> echo $ANTHROPIC_API_KEY
> ```

### 日志查看

```bash
# 查看日志
openfang logs

# 实时日志
openfang logs --follow
```

日志文件位置：
- **macOS/Linux**: `~/.openfang/logs/`
- **Windows**: `%USERPROFILE%\.openfang\logs\`

### 重置配置

```bash
# 重置为默认配置（会备份现有配置）
openfang config reset
```

## 卸载

### macOS / Linux

```bash
rm -rf ~/.openfang
rm $(which openfang)
```

### Windows

```powershell
Remove-Item -Recurse -Force "$env:USERPROFILE\.openfang"
Remove-Item -Force (Get-Command openfang).Source
```
