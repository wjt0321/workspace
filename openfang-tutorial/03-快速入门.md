# OpenFang 快速入门

## 5分钟快速上手

### 前置条件

确保已完成 [[02-安装指南|安装]] 并成功启动服务。

```bash
# 确认服务运行
openfang status
# Dashboard: http://localhost:4200
```

### 第一步：激活第一个 Hand

Hands 是 OpenFang 的核心——预构建的自主能力包。

```bash
# 查看所有可用 Hands
openfang hand list

# 激活 Researcher Hand
openfang hand activate researcher

# 查看激活状态
openfang hand status researcher
```

> [!tip] 内置 Hands
> - `researcher` - 深度研究助手
> - `lead` - 销售线索生成
> - `collector` - 情报收集
> - `clip` - 视频处理
> - `predictor` - 趋势预测
> - `twitter` - 社媒管理
> - `browser` - 网页自动化

### 第二步：与 Agent 对话

```bash
# 启动交互式对话
openfang chat researcher
```

示例对话：

```
> 分析一下AI智能体框架的发展趋势
```

## 基本操作

### Agent 管理

```bash
# 查看所有 agents
openfang agent list

# 生成预置 agent
openfang agent spawn coder

# 创建自定义 agent
openfang agent create my-agent --skill custom

# 删除 agent
openfang agent delete my-agent
```

### Hand 管理

```bash
# 激活
openfang hand activate <name>

# 暂停（保留状态）
openfang hand pause <name>

# 恢复
openfang hand resume <name>

# 停止并清除状态
openfang hand stop <name>

# 查看详细信息
openfang hand show <name>
```

### Skill 管理

```bash
# 列出所有技能
openfang skill list

# 安装技能（从 FangHub）
openfang skill install <skill-name>

# 创建自定义技能
openfang skill create my-skill

# 验证技能格式
openfang skill validate my-skill
```

### Memory 操作

```bash
# 查看会话历史
openfang memory sessions

# 搜索记忆
openfang memory search "关键词"

# 导出记忆
openfang memory export --format json

# 清除记忆
openfang memory clear --confirm
```

## 实用示例

### 示例1：每日市场研究

```bash
# 激活 Researcher 并设置定时任务
openfang hand activate researcher \
  --schedule "0 9 * * *" \
  --task "研究AI领域最新动态，生成日报"

# 查看生成的报告
openfang memory search "日报" --latest
```

### 示例2：销售线索生成

```bash
# 配置 ICP（理想客户画像）
openfang hand configure lead --icp '
  industry: ["SaaS", "FinTech"]
  company_size: "50-500"
  location: ["US", "EU"]
'

# 激活并运行
openfang hand activate lead

# 导出线索
openfang hand export lead --format csv --output leads.csv
```

### 示例3：竞品监控

```bash
# 添加监控目标
openfang hand configure collector --targets '
  - company: "CompetitorA"
    watch: ["pricing", "features", "news"]
  - company: "CompetitorB"  
    watch: ["blog", "github"]
'

# 激活监控
openfang hand activate collector

# 查看收集的情报
openfang hand report collector
```

### 示例4：视频内容制作

```bash
# 从 YouTube 创建短视频
openfang hand activate clip --input "https://youtube.com/watch?v=xxx"

# 配置输出
openfang hand configure clip --output '
  platforms: ["telegram", "whatsapp"]
  format: "vertical"
  duration: "60s"
'
```

## OpenAI 兼容 API

OpenFang 提供 OpenAI 兼容的 API 端点：

```bash
curl -X POST http://localhost:4200/v1/chat/completions \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer $OPENAI_API_KEY" \
  -d '{
    "model": "researcher",
    "messages": [
      {"role": "user", "content": "分析Q4市场趋势"}
    ],
    "stream": true
  }'
```

Python 示例：

```python
from openai import OpenAI

client = OpenAI(
    base_url="http://localhost:4200/v1",
    api_key="your-key"  # 可以是任意值
)

response = client.chat.completions.create(
    model="researcher",
    messages=[
        {"role": "user", "content": "分析AI市场趋势"}
    ],
    stream=True
)

for chunk in response:
    print(chunk.choices[0].delta.content, end="")
```
